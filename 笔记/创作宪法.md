# 🍅 番茄爆款专用·创作宪法

## 一、 核心指令（The One Thing）

**用“扫大街”的态度去“灭世”，用“养猪”的逻辑去“修仙”。**
全书只能有一种爽感来源：**极致的逼格源于极致的无知。**

## 二、 爽点公式（必须死守的节奏）

每一章的爆点段落，必须严格遵循以下 **4 步走**：

1. **起势（装X）**：反派/灾难/神器登场，特效拉满，恐怖如斯（外界视角）。
2. **破功（降维）**：主角登场，只关心“脏不脏、吵不吵、能不能吃”（主角视角）。
3. **灭杀（秒杀）**：主角用最廉价的动作（扫、拍、泼、踩）瞬间解决大恐怖（动作描写）。
4. **封神（脑补）**：旁观者（弟子/敌人）瞳孔地震，强行解释为“大道至简、返璞归真”（情绪宣泄）。

## 三、 写作红线（触之即死）

### ❌ 绝对禁止：

1. **禁止主角内心戏复杂**：主角绝不能思考“这个世界是不是不对劲”。他的大脑里只有生活琐事。
2. **禁止主角“讲道理”**：主角不要试图去理解修仙者，他要把修仙者当成**精神病、流浪汉、推销员**处理。
3. **禁止长篇大论的设定**：除非是用在反派装 X 的时候（为了马上被打脸）。
4. **禁止温吞水**：不要写“主角慢慢打扫卫生”，要写“主角暴躁地铲除了足以毁灭一城的垃圾”。

### ✅ 必须执行：

1. **情绪必须“急”**：主角要有**起床气/强迫症/抠门属性**。他解决问题不是因为正义，是因为**“烦死了”**。
2. **视觉必须“脏”**：把神兽写成癞皮狗，把神器写成烧火棍，把天材地宝写成发霉烂菜。
3. **脑补必须“疯”**：弟子的脑补不要停留在“师尊好强”，要上升到**“师尊在重演开天辟地”**的高度。

## 四、 两套语言体系（强制隔离）

### 1. 主角语料库（市井大爷风）

- **态度**：嫌弃、不耐烦、心疼钱、护犊子（护财）。
- **高频词**：晦气、真脏、瞎折腾、败家玩意儿、这啥破烂、能不能消停点。
- **示例**：“这雷劫怎么劈个没完？把我家瓦都震酥了！有没有公德心？”

### 2. 外界语料库（克苏鲁/玄幻风）

- **态度**：战栗、绝望、狂热、跪拜、不可名状。
- **高频词**：道韵、法则、寂灭、因果、大恐怖、禁忌之主、不可直视。
- **示例**：“那是……灭世雷劫？竟被他一声怒喝斥退？言出法随！这是言出法随啊！”

## 五、 章节卡片（AI 提示词模板）

> 每次生成前，必须填写此卡片，确保 AI 不跑偏。

| 字段              | 填写要求（越夸张越好）                              |
| :---------------- | :-------------------------------------------------- |
| **本章生活目标**  | 主角要做什么鸡毛蒜皮的小事？（例：腌咸菜、补裤裆）  |
| **遭遇的大恐怖**  | 谁来阻碍主角做这件小事？（例：魔尊降临、灭世洪水）  |
| **降维打击动作**  | 主角用了什么工具/动作解决？（例：用压缸石砸死魔尊） |
| **关键误会点**    | 弟子把这个动作解读成了什么？（例：镇压万古魔源）    |
| **资源/地位结算** | 主角得到了什么生活物资？外界把这当成了什么神物？    |
| **本章钩子**      | 下一章谁来送死/谁来跪舔？                           |

## 六、 验收标准（没有达到重写）

1. **有没有笑出声？** （反差够不够大？）
2. **主角是不是显得很无辜？** （越无辜越装 X）
3. **反派是不是死得很憋屈？** （铺垫了 500 字，被 10 个字秒杀）
4. **有没有出现一句“金句”？** （例：把天道之子当成查水表的）

---

### 💡 为什么要这么改？

1.  **把“解释权”改成“降维”：** “解释权”太文绉绉。番茄读者要看的是“你拼命修炼了几千年，不如我随便扔的一块砖”。
2.  **把“正常”改成“嫌弃”：** 主角光是“正常”是不够爽的。他必须**“嫌弃”**修仙界。比如反派送来一把绝世宝剑，主角嫌弃它**“没开刃，切菜费劲”**。这种**价值观的践踏**才是最爽的。
3.  **强调“急”和“躁”：** 你的原版太稳了。番茄文需要节奏快。主角要有**“别烦我，我忙着煮饭呢”**的紧迫感，这样打脸才脆。
