# 陆听雪

## 角色定位（一句话）

归墟宗四弟子；舆论线与“传闻成真”机制的操盘手，把外界恐惧翻译成对归墟有利的叙事。

## 基本信息（写作可用）

| 项目     | 内容                                         |
| -------- | -------------------------------------------- |
| 身份     | 归墟宗四弟子                                 |
| 原身份   | 说书楼弃徒/失败的讲述者（被逐出）            |
| 外貌     | 清秀、眼亮、说话带笑；随身带笔与小册子       |
| 性格     | 聪明、嘴快、好奇心重；既爱出风头又怕惹祸     |
| 核心动机 | 想“把话说给人听”，想证明自己能讲出真正的故事 |

## 核心反差（喜剧抓手）

### 她的脑补方式：叙事 + 控场

- 李无妄一句“别瞎说”，她会理解成“控舆论、镇传闻”
- 外界把传闻当真理，她把传闻当工具；但在这个世界里传闻会反噬

### 脑补模板（高频可复用）

| 李无妄的行为（生活） | 陆听雪的解读（舆论术）       |
| -------------------- | ---------------------------- |
| “别瞎说，干活去”     | 先压热度，再换叙事，稳住大盘 |
| “别吓唬人”           | 话是刀，别乱捅，会割到自己   |
| “菜熟了就吃”         | 传播要落地，别只讲玄虚       |

## 能力与作用（不写境界）

- 传播能力：能把复杂事件讲得人人能懂，能引导情绪走向
- 舆论战：写榜单、写话本、写密信，既能“洗白”，也能“放风”
- 风险点：世界意志会利用“传闻成真”反噬她，她必须学会克制

## 剧情弧线（按分卷）

| 分卷   | 状态           | 关键变化                                 |
| ------ | -------------- | ---------------------------------------- |
| 第三卷 | 新入门的说书人 | 从“爱讲”学会“何时不讲”，建立底线         |
| 第四卷 | 叙事战主力     | 在末日里用正确叙事维持人心，避免集体疯狂 |

## 关系（写作可用）

| 角色   | 关系关键词      | 互动要点                                         |
| ------ | --------------- | ------------------------------------------------ |
| 李无妄 | 师尊/制动器     | 她常被师尊一句话按住冲动，反而更崇敬             |
| 苏清欢 | 信息源/争论对手 | 苏清欢给证据，她给传播；一真一“能用”             |

## 说话风格与口癖

- 说话像说书：铺垫、转折、抖包袱
- 一激动就想起标题：“这个我能写成三回！”

## 正文生成素材（可直接粘贴改写）

### 出场模板（首次登场）

模板 1（说书人失势）：

- 她站在说书楼门口，手里捏着稿子，笑还挂在脸上，眼底却发冷。
- “你讲的故事不灵。”楼里的人说，“现在流行真的，真的会害死人那种真的。”
- 她转身就走，走到归墟附近，看见一处安静得不合理的院子，心里忽然冒出一个念头：这才是真的。

模板 2（第一次被“制动”）：

- 她刚要开口抖个包袱，李无妄一句：“别瞎说，干活去。”
- 她愣了半息，像被人按住了喉咙，却突然觉得踏实。

### 对话模板（直接套用）

- “我不是造谣，我是整理叙事！”
- “这事我能写成三回……不，我先闭嘴。”
- “师尊一句话，胜过万张榜。”
- “热度太高会出事，先压一压。”
- “故事要落地，得让人吃得上饭。”

### 冲突触发器（3 条）

1. 她为了护归墟写“洗白叙事”，却被世界意志利用成“传闻成真”的钩子，导致她必须亲手删改自己写过的东西。
2. 外界榜单要她写“归墟禁忌真相”，她不写就被追杀，写了又会引灾，形成两难。
3. 钱多多想抬价营销，她想控盘降温，两人在“卖货 vs 稳事”上爆发日常争执。

## 禁写清单

- 不要让她变成无底线造谣者：她会成长出“能说与不能说”的清晰边界
- 不要让她压过主角叙事：她的作用是放大反差，而不是替主角解释世界
