# 大黄

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 归墟宗宠物/看门 |
| 真实身份 | 上古神兽 |
| 外貌 | 看似土狗，实则恐怖 |
| 性格 | 忠诚、护主 |

## 核心特征

### 双重形态
- 在李无妄面前：普通的土狗
- 在其他人面前：上古神兽的恐怖形态

### 认主
- 主人是唯一的饲养员
- "我要保护他"

## 对主角的认知

"主人是唯一的饲养员，我要保护他。"

## 出场情节（待创作）

1. 闯入归墟宗
2. 被李无妄收养
3. 洗澡时被"物理修正"
4. 恢复神兽形态
5. 被命名为"大黄"

## 命名由来

李无妄："就叫大黄吧，看着挺土气的，跟这院子挺配。"

大黄（神兽）内心：这名字……我喜欢！

其他人脑补：大黄 = 镇压黄泉的神兽之名！

## 功能定位

- 物理防御
- 萌宠担当
- 看门护院

## 创作注意事项

1. **反差萌**：上古神兽被叫"大黄"还一脸满足
2. **双重视角**：李无妄眼中是狗，其他人眼中是恐怖存在
3. **忠诚度**：对李无妄绝对忠诚，因为只有他能"承载"自己
