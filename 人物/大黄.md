# 大黄

## 1. 核心标签（番茄爆款风）
*   **身份**：归墟看门狗 / 上古吞天兽
*   **核心人设**：**极致领地意识 / 舔狗（物理）**。
*   **脑补滤镜**：**食物链/等级压制**。认为师尊是宇宙最强捕食者，其他人都是储备粮。
*   **口头禅**：“汪！”（翻译：滚！）、“呜~”（翻译：主人我饿了）。

## 2. 误读机制（爽点来源）
*   **对主角的误解**：认为主角是唯一能降伏它的存在，主角的每一个动作都是在宣示主权。
*   **行为模式**：
    *   主角喂骨头 -> 它认为是“赐予神力” -> 吞下后体型暴涨，一口吞掉上门的魔尊。
    *   主角洗澡（给它洗） -> 它认为是“洗练血脉” -> 舒服得发出雷鸣般的呼噜声（震碎方圆百里妖兽）。
    *   主角拴绳 -> 它认为是“封印解除的保险” -> 只有带上绳子才觉得自己是只安全的狗，一旦解开就发疯。

## 3. 基础档案
*   **外貌**：平时是一只秃尾巴的中华田园犬（土黄狗），眼神清澈愚蠢；战斗形态是遮天蔽日的吞天巨兽，散发着毁灭气息。
*   **性格**：对内憨厚粘人（只对李无妄），对外凶残暴虐（对所有靠近院子的生物）。
*   **恐惧点**：**被遗弃**（怕师尊不要它了）、**没饭吃**。

## 4. 关系网
*   **对李无妄**：绝对忠诚，视为主宰。只要李无妄在场，它就是只乖狗狗；李无妄一转身，它就是地狱恶犬。
*   **对红绫**：勉强认可的“铲屎官二号”，偶尔陪她练练手。
*   **对钱多多**：讨厌的胖子，总想拔它的毛，但看在经常带好吃的份上，忍了。

## 5. 成长线（本卷）
*   **起点**：流浪到归墟门口，被李无妄一根骨头收买，从此赖着不走。
*   **拐点**：在“镇魔司围攻”事件中，一声吼叫震碎万千法宝，确立了“归墟禁地”的威名。
*   **终点**：成为修仙界公认的“禁忌守护者”，名字可止小儿夜啼，但在归墟里依然是只爱啃骨头的土狗。
