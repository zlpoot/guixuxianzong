# 李无妄

## 1. 核心标签（番茄爆款风）

- **身份**：归墟宗主 / 世界唯一的正常人
- **核心人设**：**务实的攒钱狂 / 偏执的生活家**。无论发生什么，第一反应都是“这能不能换钱？”或者“这会不会影响我种菜？”
- **表面强动机**：**存够100万两白银，去凡人城池买个三进三出的大宅子，娶个贤惠（屁股大好生养）的老婆，安度晚年**。
- **脑补滤镜**：**反向脑补（去魅）**。把修仙者看作“人傻钱多的中二病患者”，把妖魔看作“野味”或“害虫”。
- **口头禅**：“这日子没法过了”、“怎么到处都是害虫”、“要讲卫生”。

## 2. 误读机制（爽点来源）

- **对外界的误解**：认为外界是“贫困山区”，修仙者是“脑子不太好使的富二代/Cosplay爱好者”，天劫是“恶劣天气”。
- **行为模式**：
  - 看到灭世魔尊 -> 认为是野狗/野味 -> 考虑红烧还是清蒸。
  - 看到修仙大能 -> 认为是“大客户” -> 极力推销自己的“农产品”和“手工艺品（神器）”。
  - 看到天道崩塌 -> 认为是房子漏雨 -> 拿水泥（补天石）糊上。
  - 看到绝世神器 -> 认为是破铜烂铁 -> 拿去垫桌脚。

## 3. 基础档案

- **外貌**：看似平平无奇的青年，穿着粗布麻衣，但身上不仅一尘不染，还自带一种让周围环境“强制平静”的气场（他自己觉得是洗得干净）。
- **性格**：极度务实、轻微洁癖、护短（怕麻烦）、对“生活质量”有极高的要求。
- **恐惧点**：**断粮**（没米下锅）、**脏乱差**（院子被弄脏）、**噪音**（睡觉被吵醒）。

## 4. 关系网

- **对红绫**：觉得是个可怜的“疯丫头”，虽然脑子不太好使（老说胡话），但干活挺利索，就收留当个保姆吧。
- **对钱多多**：觉得是个“精明的小贩”，能帮自己把院子里的破烂（神器）卖出去换米，是个好帮手。
- **对大黄**：觉得是条“忠诚的土狗”，虽然长得有点怪（其实是上古神兽），但看门是一把好手。
- **对世界意志**：觉得是“顽固的害虫/恶劣的天气”，总想把自己的院子弄乱，必须定期清理（大扫除）。

## 5. 成长线（本卷）

- **起点**：只想在归墟安静种菜，觉得外界太乱不想出去。
- **拐点**：发现自己种的菜/用的农具在这些“中二病”眼中很值钱，决定**主动搞钱**，开启“归墟杂货铺”计划。
- **终点**：本来只想赚点养老钱，结果垄断了万界资源，成为了让诸天万界跪拜的“起源之主”，但他还在纠结离100万两还差多少。
