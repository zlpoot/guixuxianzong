# 李无妄

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 归墟宗宗主 |
| 年龄 | 不详（穿越者） |
| 外貌 | 青色布衣，洗得发白 |
| 性格 | 钝感、朴素、有轻度洁癖 |

## 核心特征

### 绝对正常
- 全程不觉醒（或极晚觉醒）
- 真诚地认为自己是普通人
- 任何诡异靠近他都会自动合理化

### 认知过滤
他眼中的世界 vs 真实世界：

| 他眼中的 | 真实世界 |
|----------|----------|
| 雾霾 | 污染雾气 |
| 大蜈蚣 | 天灾级道诡"千眼孽龙" |
| 流浪狗 | 上古神兽 |
| 皮肤病 | 高阶污染异化 |
| 耳鸣 | 精神尖啸 |
| 痤疮 | 道诡侵蚀 |

### 物理修正
- 用最普通的物理手段"修正"诡异
- 扫帚扫地 → 扫掉高阶污染
- 菜刀削皮 → 剥离道诡法则
- 石头砸"长虫" → 镇杀天灾级道诡

## 心理活动原则

### 必须纯粹
- 不能有"我这样做他们肯定会吓到"的想法
- 看到弟子吐血，想的是"这孩子上火"，而不是"我功力太深"

### 日常化思维
- 把一切诡异事物当成日常问题处理
- 例：道诡入侵 → "这章鱼成精了？正好缺个拖把。"

## 经典台词

- "这雾霾怎么又严重了？"
- "多喝热水，对皮肤好。"
- "大清早的，你跟个土豆较什么劲？"
- "用压酸菜的石头，换你点大米不过分吧？"
- "这破宗门，哪怕砸锅卖铁，也必须得招个正经厨子进来！"

## 创作注意事项

1. **钝感力**：心理活动必须纯粹，不能有"我在装傻"的暗示
2. **日常感**：无论多离谱的情况，都用日常逻辑处理
3. **反差感**：他的"普通"与他人的"脑补"形成喜剧核心
