# 双长老

## 角色定位（一句话）

宋缺与古百毒，归墟宗“技术支援组”；他们一出手就像天灾，但在师尊院子里只能按生活规则办事。

## 宋缺

### 基本信息（写作可用）

| 项目 | 内容                                   |
| ---- | -------------------------------------- |
| 身份 | 归墟宗后勤长老                         |
| 代号 | "灶神"                                 |
| 职责 | 宗门主厨                               |
| 外貌 | 身影在烟火中忽隐忽现；袖口常带焦香与灰 |
| 性格 | 讲究、执拗、嘴硬心软；对“火候”近乎偏执 |

### 核心特征（固定看点）

#### 火候失控（结果比人强）

- 他做饭时，火像在“审判”食材
- 成品经常呈现“法则洗礼”的效果：补到离谱、辣到顿悟、香到失语
- 他越想做正常饭，越容易做出“会把外界吓疯”的东西

#### 对李无妄的认知（自洽理由）

“宗主是唯一能把我的菜当饭吃的人。”（在他眼里这不是夸张，是事实）

### 高复用场景

| 场景                | 李无妄视角     | 外界视角          |
| ------------------- | -------------- | ----------------- |
| 做饭翻车            | 火太大/盐放多  | 道火炼世/以味定法 |
| 师尊评价“淡了/糊了” | 真实的家常挑刺 | 天命裁决/大道箴言 |
| 被要求“别熏到菜”    | 后厨纪律       | 一句话镇压灾厄    |

## 古百毒

### 基本信息（写作可用）

| 项目 | 内容                                     |
| ---- | ---------------------------------------- |
| 身份 | 归墟宗炼丹长老                           |
| 代号 | "炉鬼"                                   |
| 职责 | 炼丹/装备来源                            |
| 外貌 | 面容被药雾遮掩；走到哪儿哪儿都有刺鼻药味 |
| 性格 | 冷、轴、爱钻研；把“失败”当成数据         |

### 核心特征（固定看点）

#### 结果失控（炸炉即神通）

- 炼丹必炸，炸得越狠效果越离谱
- 爆炸常伴随“范围异象”：地面开花、影子错位、语言短暂失效
- 他本人很认真地把这些当作“副作用”，弟子与外界当作“天罚/禁术”

#### 对李无妄的认知（自洽理由）

“宗主的院子是唯一的安全区。”（他认为这是客观现象，不是吹捧）

### 高复用场景

| 场景            | 李无妄视角     | 外界视角           |
| --------------- | -------------- | ------------------ |
| 炼丹炸了        | “又把锅炸了？” | 炉鬼开天，灾厄降临 |
| 师尊递水/让通风 | 灭火、散味     | 平息天灾、澄清因果 |
| 炼出“残次品”    | 丢了可惜       | 人人疯抢的禁忌神物 |

## 双长老关系

### 关系定位

- 竞争共生，严禁暧昧
- 互怼是职业习惯，默契来自“只有你能理解我的失控”

### 互动模式

- 专业互怼："你的火太弱了" vs "你的药太臭了"
- 羁绊："只有你能承受我的灾难"

### 竞争内容

- 争“谁更有用”（后勤 vs 炼丹）
- 争“谁更不添乱”（谁先被师尊赶去院子外）
- 争“宗主先夸谁一句”（一句家常夸奖能让他们激动半个月）

### 出场情节（待创作）

1. 因"地盘之争"在宗门外引发灾难
2. 李无妄路过皱眉："烟熏到我的菜了。"
3. 两人瞬间停手
4. 争相加入归墟宗赎罪
5. 弟子们认为：宗主一句话镇压了两位灾难巨头

---

## 创作注意事项

1. 他们的强不是“境界压人”，而是“做什么都像天灾”
2. 关系界限必须清晰：专业互怼、战友共生，不写暧昧
3. 他们越想认真做事，越容易制造大场面；师尊用一句家常话就能按住

## 正文生成素材（可直接粘贴改写）

### 出场模板（首次登场）

模板 1（地盘之争引灾）：

- 山风里夹着两种味道：一股像焦糖，一股像腐药。
- 一边火光把云烧得发红，一边药雾把月色熏得发绿。
- 宋缺骂古百毒“把味做脏了”，古百毒回他“你把火当命了”，两人的争吵像天灾对轰。
- 李无妄路过，皱眉一句：“烟熏到我的菜了。”
- 下一刻，天地忽然安静，只剩锅里咕嘟一声。

### 对话模板（直接套用）

宋缺：

- “火候才是根本，别拿臭药糟蹋味。”
- “宗主说淡了？……我改。”
- “你离我的灶远点。”

古百毒：

- “数据不够，再试一次。”
- “副作用可控。”
- “你把火收一收，别把院子烤裂。”

合体互怼：

- “你先别炸。”
- “你先别烧。”
- “宗主在院里，你想死别拉我。”

### 冲突触发器（3 条）

1. 两人为了证明“谁更有用”，分别做出“正常版”的饭与丹，结果都翻车成禁忌现象，逼师尊出面用一句家常话按住。
2. 外界势力上门求药/求饭，把他们当神供着，反而惹恼两人：他们只认师尊的“能吃/不能吃、能用/不能用”。
3. 一次炸炉/一次走火把归墟外缘点燃成灾，红绫准备拔剑“清场”，钱多多准备赔钱“灭火”，形成三线同场混乱。
