# 双长老

## 概述

宋缺与古百毒，归墟宗的"灾难双星"，关系为竞争共生。

---

## 宋缺

### 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 归墟宗后勤长老 |
| 代号 | "灶神" |
| 职责 | 宗门主厨 |
| 外貌 | 身影在烟火中忽隐忽现 |

### 核心特征

#### 火候失控
- 追求极致道火
- 成品往往是法则洗礼
- 做的菜蕴含恐怖力量

#### 对主角的认知
"宗主是唯一美食家，能消化我的'法则料理'。"

### 功能定位
- 状态 Buff
- 生活喜剧
- 宗门主厨

---

## 古百毒

### 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 归墟宗炼丹长老 |
| 代号 | "炉鬼" |
| 职责 | 炼丹/装备来源 |
| 外貌 | 面容被药雾遮掩 |

### 核心特征

#### 结果失控
- 炼丹必炸
- 炸即神通
- 爆炸产生范围伤害

#### 对主角的认知
"宗主是唯一安全区，只有他能承受我的炸炉。"

### 功能定位
- 范围伤害
- 装备来源
- 灾难制造

---

## 双长老关系

### 关系定位
- **竞争共生**，严禁暧昧
- 只有战友般的共生，而非爱情

### 互动模式
- 专业互怼："你的火太弱了" vs "你的药太臭了"
- 羁绊："只有你能承受我的灾难"

### 竞争内容
- 争相加入归墟宗赎罪
- 争"谁更有用"
- 争"宗主吃谁的残渣"

### 出场情节（待创作）

1. 因"地盘之争"在宗门外引发灾难
2. 李无妄路过皱眉："烟熏到我的菜了。"
3. 两人瞬间停手
4. 争相加入归墟宗赎罪
5. 弟子们认为：宗主一句话镇压了两位灾难巨头

---

## 创作注意事项

1. **形象模糊化**：不写皱纹白发，写"身影在烟火中忽隐忽现"
2. **关系界限**：严禁暧昧描写，保持专业互怼
3. **灾难美学**：他们的"失控"是看点，不是累赘
