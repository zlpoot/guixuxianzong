# 钱多多

## 1. 核心标签（番茄爆款风）

- **身份**：归墟首席商务官（CBO） / 师尊的“白手套”
- **核心人设**：**坚定的商业信徒**。认为师尊不是在修仙，而是在下一盘名为“垄断万界”的大棋。
- **脑补滤镜**：**商业思维**。师尊的一举一动都是“市场信号”。
- **口头禅**：“这是蓝海啊！”、“师尊这是在做空天道！”、“这波红利必须吃下！”

## 2. 误读机制（爽点来源）

- **对主角的误解**：认为师尊是“万界第一资本大鳄”，手里掌握着名为“归墟”的终极产业链。
- **行为模式**：
  - 主角扔垃圾（烂菜叶） -> 他认为是“限量版神药投放” -> 搞饥饿营销，拍卖出天价。
  - 主角想娶媳妇（要钱） -> 他认为是“融资扩张信号” -> 疯狂变卖万宝阁资产来入股“归墟集团”。
  - 主角嫌吵（杀人） -> 他认为是“恶意收购的防御性打击” -> 配合师尊吞并对方宗门的产业。

## 3. 基础档案

- **外貌**：锦衣玉带，身材微胖，一脸和气生财的笑容，手里总是盘着两个核桃（其实是某种神兽的蛋）。
- **性格**：精明算计、长袖善舞、唯利是图（但师尊的利是最大的利）、极度怕死。
- **恐惧点**：**破产**（失去万宝阁继承权）、**被师尊逐出师门**（失去最大的靠山）。

## 4. 关系网

- **对李无妄**：视为最大的“金大腿”，认为只要抱紧师尊，万宝阁就能统御修仙界经济命脉。
- **对红绫**：觉得她是“赔钱货”（只会打架不赚钱），但为了讨好师尊，经常自掏腰包给她买装备。
- **对大黄**：经常试图用美食贿赂大黄，想弄几根狗毛（神兽毛）去卖。

## 5. 成长线（本卷）

- **起点**：被家族竞争对手追杀，带着万宝阁资源投奔归墟，被李无妄用“压缸石”砸死毒龙所震撼。
- **拐点**：被任命为“废品处理专员”，发现师尊扔的“废品”都是绝世神器，悟出“废物利用”的暴利模式。
- **终点**：建立了跨越诸天的“归墟物流”和“神级废品回收站”，把修仙界变成了师尊的原材料供应基地和产品倾销市场。
