# 钱多多

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 归墟宗二弟子 |
| 原身份 | 万宝阁少东家 |
| 外貌 | 圆滚滚的胖子，浑身挂满法宝 |
| 性格 | 商人本色、脑补帝、老饕 |

## 核心特征

### 脑补帝二号
- 顶级老饕，深谙食道理论
- 嘴强王者，实操废材

### 商人本能
- 万宝阁少东家
- 家族在三十六州域有分阁
- 年营业额约三亿上品灵石

### 装备
- 极品金丝软甲法袍
- 十根手指戴满储物戒指
- 脖子上挂满极品护身玉佩
- 本命乾坤戒（历代阁主才能开启）

## 对主角的认知

"宗主在布局天下，废品皆是神器。"

## 脑补案例

| 李无妄的行为 | 钱多多的解读 |
|--------------|--------------|
| 给"猛虎下山图" | 蕴含白虎真意的无上道图 |
| 把石头当添头 | 镇海神石，镇压万古的无上大印 |
| 说买"青瓦" | 教导做人要低调朴素 |
| 让他负责采购 | 把他当心腹 |

## 厨艺水平

- 理论一套一套的
- 用极品丹炉炒菜
- 用珍贵灵材当调料
- 做出来的红烧肉蕴含恐怖灵气

## 经典台词

- "要命了！要命了！"
- "师尊！您这一张'装饰'，买下十个万宝阁都绰绰有余啊！"
- "弟子钱多多，自幼对商贾之道略有心得！愿倾尽万宝阁所有家底，拜入师尊门下！"

## 创作注意事项

1. **商人本色**：即使脑补，也会下意识计算价值
2. **老饕设定**：对美食有执念，但实操是灾难
3. **与红绫的竞争**：争宠暗战是重要看点
