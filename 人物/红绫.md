# 红绫

## 1. 核心标签（番茄爆款风）
*   **身份**：归墟大弟子 / 前朝公主（被追杀者）
*   **核心人设**：**极致暴力狂 / 脑补帝一号**。
*   **脑补滤镜**：**剑道/杀伐**。把师尊的一切行为都解读为“剑意”或“无上杀招”。
*   **口头禅**：“师尊之意，是要斩尽杀绝！”、“此乃无上剑道！”

## 2. 误读机制（爽点来源）
*   **对主角的误解**：认为师尊是隐世不出的绝世剑仙，每一个动作都在演练剑法。
*   **行为模式**：
    *   主角倒水（想让她喝水） -> 她认为是“剑意化水，洗筋伐髓” -> 当场跪下顿悟，剑气冲天。
    *   主角扫地（嫌脏） -> 她认为是“横扫诸天，荡平妖魔” -> 拔剑把上门的敌人砍成碎片。
    *   主角叹气（嫌米贵） -> 她认为是“天道不公，当斩！” -> 提剑要去灭了涨价的商会（被钱多多拦住）。

## 3. 基础档案
*   **外貌**：红衣胜火，背负巨剑（其实是主角的门栓），眼神凌厉，浑身散发着生人勿进的煞气。
*   **性格**：杀伐果断、极度护短、对师尊盲目崇拜、对敌人毫不留情。
*   **恐惧点**：**师尊失望**（怕师尊觉得她悟性不够，不教她“剑法”了）。

## 4. 关系网
*   **对李无妄**：如神明般敬畏，认为师尊的每一句话都是真理，愿意为师尊扫平一切障碍（物理）。
*   **对钱多多**：觉得他满身铜臭，但既然师尊留下了，就勉强当个“管账的”保护起来。
*   **对大黄**：尊称为“护宗神兽”，经常与其切磋（单方面被虐）。

## 5. 成长线（本卷）
*   **起点**：被追杀至濒死，误入归墟，被李无妄一杯热水救活，从此死心塌地。
*   **拐点**：在“宗门大比”中，用师尊教的“切菜刀法”击败各路天骄，名震天下。
*   **终点**：成为修仙界闻风丧胆的“血衣剑仙”，但回到归墟就是个乖巧的“洗菜丫头”。
