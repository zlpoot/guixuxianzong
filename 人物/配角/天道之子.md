# 天道之子

## 角色定位（一句话）

世界意志的“对照物”与执行端；用绝对逻辑证明李无妄是异常，从而触发全界的解释权之争。

## 基本信息（写作可用）

| 项目     | 内容                                               |
| -------- | -------------------------------------------------- |
| 身份     | 世界意志化身（对外名义：天道之子/校验使）          |
| 外貌     | 端正到近乎“标准答案”；衣冠整洁、眼神无波           |
| 性格     | 冷静、克制、绝对自洽；不嘲讽、不暴怒，但会持续纠错 |
| 核心动机 | 维护“异常自洽”的世界规则，清除“正常病毒”           |

## 核心机制（写作抓手）

### 绝对逻辑（校验器）

- 他不是修士风格的“强”，而是规则层面的“对/错”
- 他会用一切可被证明的方式逼出矛盾：定义、证据、推演、复现
- 他的语言与行为要像在做“审计/验算”，而不是在打架

### 对李无妄的判断（固定论点）

- “此人行为过于正常，正常到破坏世界的异常自洽”
- “若人人以生活逻辑解释异常，世界将失去维持恐惧的结构”

## 与主角的对抗方式（生活化道争）

| 天道之子做什么          | 李无妄怎么回应           | 外界怎么理解           |
| ----------------------- | ------------------------ | ---------------------- |
| 抛出“世界虚妄/一切皆幻” | “菜熟了得吃，饿了就会死” | 真实击穿虚无，逻辑崩盘 |
| 设立“合理”试炼与规则    | “你别折腾，影响我干活”   | 师尊一句话改写校验标准 |
| 以舆论与传闻围剿        | “别瞎传，容易吓到人”     | 以凡言镇传闻成真       |

## 剧情弧线（按分卷）

| 分卷     | 状态       | 关键变化                                         |
| -------- | ---------- | ------------------------------------------------ |
| 第二卷末 | 降临与公告 | 发布“不得接触归墟”的校验令，制造对立             |
| 第三卷   | 主要对手   | 不断提出“合理证明”，反而被主角的生活逻辑反复击穿 |
| 第四卷   | 失效与退场 | 作为世界意志的工具逐渐失去解释力，为终局让路     |

## 说话风格与口癖

- 句子短、定义明确、很少比喻
- 常用词：校验、偏差、样本、复现、无效、归零

## 正文生成素材（可直接粘贴改写）

### 出场模板（首次登场）

模板 1（公告式降临）：

- 天色像被一只手抹平，风声忽然规整得像尺子量过。
- 他站在城楼前，声音不大，却让每个人都听见：“发布校验令：不得接触归墟。”
- 有人想问原因，他只补一句：“偏差不可扩散。”

模板 2（审计式交锋）：

- 他不拔剑，也不结印，只拿出一卷纸。
- “定义：正常。定义：异常。定义：可复现。”
- “请你证明，你不是偏差。”

### 对话模板（直接套用）

- “无效。”
- “样本不足，继续。”
- “偏差扩大，启动纠错。”
- “复现失败，结论保持。”
- “归零是最温和的处理方式。”

### 冲突触发器（3 条）

1. 他把归墟当“系统漏洞”公开标注，逼各方势力选边站队，引发全界解释权对撞。
2. 他用“合理证明”逼迫弟子自证：越解释越掉进语言陷阱，只有师尊的生活回应能打断流程。
3. 他与陆听雪的舆论交锋：她改叙事，他改定义；一边控场，一边校验，形成第三卷的“规则战”。

## 禁写清单

- 不要把他写成中二反派：他要像“系统提示/审计官”，冷而精确
- 不要让他靠单纯武力碾压：他的压迫来自“他能把你的叙事判成错误”
