# 隔壁村王大爷（王富贵）

## 1. 核心标签

- **身份**：归墟山脚下“牛家村”的孤寡老人 / 李无妄的唯一“酒友”。
- **核心人设**：**真·凡人**。在这个修仙世界里，他是除了李无妄之外，唯一一个没有灵根、没有修为、甚至有点老年痴呆的普通老头。
- **功能**：**认知锚点 / 吐槽担当**。

## 2. 角色作用

- **打破迪化闭环**：当修仙者们对着李无妄的锄头顶礼膜拜时，王大爷会走过来说：“李后生，你这锄头都生锈了，俺家有把新的，借你使使？” —— 瞬间把逼格拉回地面，制造尴尬的喜剧效果。
- **强化主角凡人认知**：李无妄通过与王大爷的对比，确认自己确实是个凡人（虽然他比王大爷强亿点点，但他觉得是因为王大爷太老了）。

## 3. 经典台词

- “李后生，你家这些亲戚（修仙大能）咋都爱磕头呢？是不是腿脚不好？”
- “李后生，你这‘猫’（吞天蟒）咋没腿？看着跟俺酒缸里泡的那条长虫（蛇）一样？要不给你剁了泡酒？”
- “啥？那个飞在天上的（元婴老怪）？那是放风筝的吧？”

## 4. 互动模式

- **蹭饭**：经常提着一瓶劣质烧酒来找李无妄蹭饭，李无妄嫌弃他的酒难喝，但还是会拿出“神级下酒菜（龙肝凤髓）”招待他。
- **吹牛**：两人经常蹲在田埂上吹牛，王大爷吹自己年轻时见过县太爷，李无妄吹自己以后要娶个屁股大的老婆。旁边的修仙者听得瑟瑟发抖，以为他们在讨论“统治诸天”的大计。
