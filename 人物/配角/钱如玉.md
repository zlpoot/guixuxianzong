# 钱如玉

## 角色定位（一句话）

万宝阁掌权者；理性型脑补的“资源发动机”，负责把归墟从破院子养成能撑过末日的据点。

## 基本信息（写作可用）

| 项目 | 内容                                           |
| ---- | ---------------------------------------------- |
| 身份 | 万宝阁阁主 / 钱多多的姐姐                      |
| 年龄 | 比钱多多大三岁                                 |
| 外貌 | 身材高挑，气质冷艳，典型的御姐                 |
| 性格 | 精明、强势、护短、外冷内热                     |
| 称号 | "玉罗刹"（商界称呼）/ "姐姐"（钱多多专属噩梦） |

## 核心特征

### 商界女王

- 万宝阁的实际掌权者
- 精于算计，从不出错
- 商界无人敢惹的"玉罗刹"

### 弟控属性

- 对钱多多极其护短
- 但表达方式是"管教"
- 钱多多最怕的人就是她

### 与红绫的对比

| 项目           | 红绫           | 钱如玉           |
| -------------- | -------------- | ---------------- |
| 类型           | 少女型         | 御姐型           |
| 性格           | 热血、冲动     | 冷静、精明       |
| 对李无妄的态度 | 崇拜           | 怀疑→认可→敬畏   |
| 战斗风格       | 剑修，正面硬刚 | 法宝流，以多打少 |
| 与钱多多关系   | 竞争对手       | 姐弟（压制关系） |

## 对主角的认知

### 初期（怀疑）

"我弟弟怎么突然拜了一个野人为师？这人什么来头？"

### 中期（调查）

"不对劲，这个人太'正常'了，正常得不正常。"

### 后期（敬畏）

"宗主深不可测……我弟弟果然有眼光。"

### 最终（脑补）

"宗主的'正常'，是对大道的终极领悟！"

## 写作抓手（与红绫区分）

- 她的脑补更像“商业尽调”：先怀疑，再验证，再押注，再全力支持
- 她的敬畏更克制：不跪也不喊口号，但会把资源与规则摆出来护住归墟
- 她能把外界势力的“恐惧”翻译成“合作条款”，让归墟受益而李无妄不烦

## 与钱多多的关系

### 日常互动

- 钱多多：姐姐，我拜师了！
- 钱如玉：什么师？给我回来！
- 钱多多：不回！师尊对我很好！
- 钱如玉：……（杀向归墟宗）

### 姐弟相处模式

- 钱多多在外面是少东家，在姐姐面前是弟弟
- 钱如玉对钱多多有绝对的压制力
- 但她也是最爱护钱多多的人

## 与归墟宗的关系

### 发展轨迹

1. 得知钱多多拜师，怒气冲冲杀向归墟宗
2. 被李无妄的"正常"震慑
3. 开始怀疑人生
4. 最终也加入"脑补大军"
5. 成为归墟宗的"编外后勤部长"

### 功能定位

- 万宝阁的资源支持
- 外界势力的"代言人"
- 与红绫形成"少女vs御姐"的对比
- 钱多多的"克星"

## 经典桥段

### 杀向归墟宗

- 钱如玉带着万宝阁精锐杀到归墟外围
- 结果被归墟的污染逼退
- 钱多多出来接应，一脸得意："姐姐，师尊说让你自己进来。"
- 钱如玉：他在考验我！
- 李无妄（内心）：这胖子怎么又带人来蹭饭了？

### 与李无妄的对话

- 钱如玉：阁下是什么人？为何收我弟弟为徒？
- 李无妄：他非要拜师，我就收了。对了，你带钱了吗？
- 钱如玉：……？
- 李无妄：下次来记得带点米，家里快断粮了。
- 钱如玉（内心）：他在暗示我，归墟宗不缺钱，缺的是"根基"？他在考验我的悟性！

### 与红绫的"暗战"

- 两人都认为自己更了解"师尊"
- 红绫：我是大弟子，我更懂师尊！
- 钱如玉：我经营万宝阁，我更懂布局！
- 李无妄：你们俩能不能别在我门口吵架？影响我晒太阳。

模板 1（杀到归墟外围）：

- 她带队停在归墟外缘，身后是万宝阁最精锐的护卫，前方是连风都不肯靠近的雾。
- 她看见自己的部下开始失语，手指开始发抖，脸色却一点没变。
- 她只说一句：“把价码准备好。人可以跪，账不能乱。”

模板 2（第一次“尽调式脑补”）：

- 她见到李无妄时，先看院子：瓦、墙、锅、米袋、菜畦。
- 她听见对方问：“你带钱了吗？”
- 她停了半息，心里却迅速下了结论：不谈玄学，只谈米盐，这种人最可怕。

### 冲突触发器（3 条）

1. 她用商业规则压外界势力的恐惧：谈合作、定条款、立规矩，结果被误读为“归墟立国”，引发更大围观。
2. 她与红绫争夺解释权：红绫靠信仰，她靠尽调；争到最后师尊一句“别吵”，两人同时当成天命各自执行。
3. 她整顿万宝阁供应链以支援归墟，触动旧势力利益，被迫打一次“商战+暗杀+舆论”的复合战。

## 创作注意事项

1. **御姐人设**：保持冷艳、强势的形象，但在李无妄面前逐渐"破防"
2. **弟控属性**：对钱多多的爱要体现在行动上，不是嘴上
3. **脑补过程**：她的脑补要更"理性"，与红绫的"感性脑补"形成对比
4. **与红绫的关系**：从敌对到认可，形成"姐妹"般的互动
5. **别让她变成工具人**：她的决策要有代价与风险，且她会为“弟弟”承担外界压力
