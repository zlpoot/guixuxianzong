# 钱如玉

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 万宝阁阁主 / 钱多多的姐姐 |
| 年龄 | 比钱多多大三岁 |
| 外貌 | 身材高挑，气质冷艳，典型的御姐 |
| 性格 | 精明、强势、护短、外冷内热 |
| 称号 | "玉罗刹"（商界称呼）/ "姐姐"（钱多多专属噩梦） |

## 核心特征

### 商界女王
- 万宝阁的实际掌权者
- 精于算计，从不出错
- 商界无人敢惹的"玉罗刹"

### 弟控属性
- 对钱多多极其护短
- 但表达方式是"管教"
- 钱多多最怕的人就是她

### 与红绫的对比

| 项目 | 红绫 | 钱如玉 |
|------|------|--------|
| 类型 | 少女型 | 御姐型 |
| 性格 | 热血、冲动 | 冷静、精明 |
| 对李无妄的态度 | 崇拜 | 怀疑→认可→敬畏 |
| 战斗风格 | 剑修，正面硬刚 | 法宝流，以多打少 |
| 与钱多多关系 | 竞争对手 | 姐弟（压制关系） |

## 对主角的认知

### 初期（怀疑）
"我弟弟怎么突然拜了一个野人为师？这人什么来头？"

### 中期（调查）
"不对劲，这个人太'正常'了，正常得不正常。"

### 后期（敬畏）
"宗主深不可测……我弟弟果然有眼光。"

### 最终（脑补）
"宗主的'正常'，是对大道的终极领悟！"

## 与钱多多的关系

### 日常互动
- 钱多多：姐姐，我拜师了！
- 钱如玉：什么师？给我回来！
- 钱多多：不回！师尊对我很好！
- 钱如玉：……（杀向归墟宗）

### 姐弟相处模式
- 钱多多在外面是少东家，在姐姐面前是弟弟
- 钱如玉对钱多多有绝对的压制力
- 但她也是最爱护钱多多的人

## 与归墟宗的关系

### 发展轨迹
1. 得知钱多多拜师，怒气冲冲杀向归墟宗
2. 被李无妄的"正常"震慑
3. 开始怀疑人生
4. 最终也加入"脑补大军"
5. 成为归墟宗的"编外后勤部长"

### 功能定位
- 万宝阁的资源支持
- 外界势力的"代言人"
- 与红绫形成"少女vs御姐"的对比
- 钱多多的"克星"

## 经典桥段

### 杀向归墟宗
- 钱如玉带着万宝阁精锐杀到归墟外围
- 结果被归墟的污染逼退
- 钱多多出来接应，一脸得意："姐姐，师尊说让你自己进来。"
- 钱如玉：他在考验我！
- 李无妄（内心）：这胖子怎么又带人来蹭饭了？

### 与李无妄的对话
- 钱如玉：阁下是什么人？为何收我弟弟为徒？
- 李无妄：他非要拜师，我就收了。对了，你带钱了吗？
- 钱如玉：……？
- 李无妄：下次来记得带点米，家里快断粮了。
- 钱如玉（内心）：他在暗示我，归墟宗不缺钱，缺的是"根基"？他在考验我的悟性！

### 与红绫的"暗战"
- 两人都认为自己更了解"师尊"
- 红绫：我是大弟子，我更懂师尊！
- 钱如玉：我经营万宝阁，我更懂布局！
- 李无妄：你们俩能不能别在我门口吵架？影响我晒太阳。

## 经典台词

- "钱多多，给我滚回来！"
- "这个人……不简单。"（经典脑补开场白）
- "宗主的意思是……"（开始脑补）
- "万宝阁从今日起，全力支持归墟宗。"（被同化）
- "师妹，你错了，师尊的深意是这样的……"（与红绫争论）

## 创作注意事项

1. **御姐人设**：保持冷艳、强势的形象，但在李无妄面前逐渐"破防"
2. **弟控属性**：对钱多多的爱要体现在行动上，不是嘴上
3. **脑补过程**：她的脑补要更"理性"，与红绫的"感性脑补"形成对比
4. **与红绫的关系**：从敌对到认可，形成"姐妹"般的互动
