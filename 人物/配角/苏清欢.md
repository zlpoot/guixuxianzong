# 苏清欢

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 外界"真相调查员" / 归墟宗编外人员 |
| 原身份 | 九大仙门之一"天机阁"的情报司首座 |
| 外貌 | 白衣胜雪，清冷出尘，眼神锐利 |
| 性格 | 理性、执着、不服输、有点强迫症 |
| 称号 | "清醒者"（自封）/ "被迷惑最深的人"（他人认为） |

## 核心特征

### 唯一的清醒者
- 她是唯一一个真正看穿李无妄"只是普通人"的人
- 但她的结论没人相信
- 每次试图揭露真相，都被当成"被宗主考验"

### 调查员本能
- 天机阁情报司出身，擅长分析推理
- 对"归墟宗传说"产生了强烈怀疑
- 决定亲自调查，结果越调查越崩溃

### 与红绫的对比

| 项目 | 红绫 | 苏清欢 |
|------|------|--------|
| 对李无妄的认知 | 宗主是无上大能 | 他就是个普通人 |
| 行为模式 | 脑补一切 | 分析一切 |
| 结果 | 被认为"悟性高" | 被认为"被考验" |
| 心态 | 崇拜 | 崩溃 |

## 对主角的认知

"他就是个普通人！你们为什么都不信我？！"

## 经典桥段

### 调查归墟宗
- 她潜入归墟外围，观察了三天三夜
- 结论：李无妄的所有行为都是普通人的日常
- 她兴奋地向外公布真相
- 结果：修仙界认为她"被归墟宗主考验，需要悟出真谛"

### 与红绫的争论
- 苏清欢："那杯就是白开水！我亲眼看见他从水缸里舀的！"
- 红绫："师姐，你还没悟透。师尊是在告诉我们，大道至简，最平凡的水才是最珍贵的。"
- 苏清欢："……"（崩溃）

### 被李无妄"点拨"
- 李无妄随口说了一句："你看着挺聪明的，怎么老皱着眉头？想开点。"
- 苏清欢：他是在暗示我"放下执念"？
- 其他人：宗主在点拨她！她在接受传承！
- 苏清欢：我就是想证明你是普通人啊！！！

## 功能定位

- 喜剧担当：她的崩溃是重要笑点
- 对照组：与脑补帝们形成对比
- 真相视角：让读者看到"真相是什么"
- 桥梁角色：连接归墟宗与外界

## 出场时机

- 建议在第二卷初登场
- 作为"外界调查员"潜入归墟
- 逐渐被"同化"（不是真的同化，而是被迫接受现状）

## 经典台词

- "他就是个普通人！"
- "你们为什么都不信我？！"
- "这杯水……就是水啊！"
- "我调查了三年，结论是——他真的只是在种菜。"
- "算了，累了，毁灭吧，我加入归墟宗。"

## 创作注意事项

1. **她是对的**：她的分析永远是正确的，但没人信
2. **崩溃美学**：她的崩溃要有层次感，从愤怒到无奈到放弃抵抗
3. **不黑化**：她不会因没人相信而黑化，只会越来越"摆烂"
4. **与红绫的互动**：两人是"脑补vs分析"的绝佳对照
