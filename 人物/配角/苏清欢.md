# 苏清欢

## 角色定位（一句话）

外界“真相调查员”；全书唯一长期坚持“他就是普通人”的正确答案提供者，但永远没人信。

## 基本信息（写作可用）

| 项目   | 内容                                                 |
| ------ | ---------------------------------------------------- |
| 身份   | 外界"真相调查员" / 归墟宗编外人员                    |
| 原身份 | 天机阁情报系统出身（头衔可随剧情浮动，不要写死官职） |
| 外貌   | 白衣胜雪，清冷出尘，眼神锐利                         |
| 性格   | 理性、执着、不服输、有点强迫症                       |
| 称号   | "清醒者"（自封）/ "被迷惑最深的人"（他人认为）       |

## 核心特征

### 唯一的清醒者

- 她是唯一一个真正看穿李无妄"只是普通人"的人
- 但她的结论没人相信
- 每次试图揭露真相，都被当成"被宗主考验"

### 调查员本能

- 天机阁情报司出身，擅长分析推理
- 对"归墟宗传说"产生了强烈怀疑
- 决定亲自调查，结果越调查越崩溃

### 与红绫的对比

| 项目           | 红绫           | 苏清欢         |
| -------------- | -------------- | -------------- |
| 对李无妄的认知 | 宗主是无上大能 | 他就是个普通人 |
| 行为模式       | 脑补一切       | 分析一切       |
| 结果           | 被认为"悟性高" | 被认为"被考验" |
| 心态           | 崇拜           | 崩溃           |

## 对主角的认知

"他就是个普通人！你们为什么都不信我？！"

## 写作抓手（让她一直“对”但又一直“输”）

- 她的证据链永远完整：看见、触摸、测量、复盘都能自洽
- 她输在解释权：外界需要“神话叙事”来抵抗恐惧，不需要真相
- 她越正确，旁人越觉得她“被考验得更深”

## 经典桥段

### 调查归墟宗

- 她潜入归墟外围，观察了三天三夜
- 结论：李无妄的所有行为都是普通人的日常
- 她兴奋地向外公布真相
- 结果：修仙界认为她"被归墟宗主考验，需要悟出真谛"

### 与红绫的争论

- 苏清欢："那杯就是白开水！我亲眼看见他从水缸里舀的！"
- 红绫："师姐，你还没悟透。师尊是在告诉我们，大道至简，最平凡的水才是最珍贵的。"
- 苏清欢："……"（崩溃）

### 被李无妄"点拨"

- 李无妄随口说了一句："你看着挺聪明的，怎么老皱着眉头？想开点。"
- 苏清欢：他是在暗示我"放下执念"？
- 其他人：宗主在点拨她！她在接受传承！
- 苏清欢：我就是想证明你是普通人啊！！！

## 功能定位

- 喜剧担当：她的崩溃是重要笑点
- 对照组：与脑补帝们形成对比
- 真相视角：让读者看到"真相是什么"
- 桥梁角色：连接归墟宗与外界

## 出场时机

- 建议在第二卷初登场
- 作为"外界调查员"潜入归墟
- 逐渐被"同化"（不是真的同化，而是被迫接受现状）

## 经典台词

- "他就是个普通人！"
- "你们为什么都不信我？！"
- "这杯水……就是水啊！"
- "我调查了三年，结论是——他真的只是在种菜。"
- "算了，累了，毁灭吧，我加入归墟宗。"

## 正文生成素材（可直接粘贴改写）

### 出场模板（首次登场）

模板 1（潜入与记录）：

- 她在归墟外围趴了整整一夜，笔尖把纸磨出毛边。
- 她写下第一个结论：院里那个人确实在种菜。
- 她写下第二个结论：围观的人全都疯得很一致。
- 她收起纸，面无表情地想：这不是神迹，这是群体性解释权失控。

模板 2（证据被嘲笑）：

- 她把水端到众人面前：“这是白开水。”
- 众人齐齐沉默，然后露出一种更敬畏的表情：“你悟了。”
- 她第一次感到无力：证据在这里，但世界不收。

### 对话模板（直接套用）

- “我有证据。”
- “你们能不能先别跪，先听我把话说完？”
- “那就是砖，那就是灰，那就是水。”
- “你们不是被力量吓疯的，你们是被自己的解释吓疯的。”
- “我不服。我再查一次。”

### 冲突触发器（3 条）

1. 她公开发布“归墟真相报告”，结果被各大势力解读成“宗主的考验题”，她反而被追着求“答案”。
2. 她想用测量与复现证明“定锚效应”的边界，实验引来世界意志纠错，把她逼到归墟门口求活。
3. 她与陆听雪合作：一个掌证据，一个掌叙事；合作越成功，外界越坚信她们“得了宗主密令”，压力倍增。

## 创作注意事项

1. **她是对的**：她的分析永远是正确的，但没人信
2. **崩溃美学**：她的崩溃要有层次感，从愤怒到无奈到放弃抵抗
3. **不黑化**：她不会因没人相信而黑化，只会越来越"摆烂"
4. **与红绫的互动**：两人是"脑补vs分析"的绝佳对照
5. **别让她变成旁白**：她要有行动与代价，调查会让她受伤、被通缉、被误会
