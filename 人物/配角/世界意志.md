# 世界意志（天道校验器）

## 角色定位（一句话）

终极反派；不是“人”，而是维护异常自洽的校验系统，把李无妄的正常当成必须清除的病毒。

## 核心职责（写作抓手）

- 维护“异常即常态”的世界叙事，使污染、献祭、仪式、恐惧都能自圆其说
- 当出现“正常锚点”导致异常失效时，启动纠错、升级打压、最终重启

## 行为升级路径（按大纲）

| 阶段       | 手段                             | 读者可见表现                                      |
| ---------- | -------------------------------- | ------------------------------------------------- |
| 天灾试探   | 暴雨、落雷、地裂等外力           | 归墟周围出现不合理天象，但被师尊当成“收衣服/堵漏” |
| 认知污染   | 传闻成真、物品长嘴、逻辑自相矛盾 | 世界用“故事”攻击人心，让人先疯再跪                |
| 制造对照物 | 天道之子登场                     | 用绝对逻辑逼迫世界选择：信生活还是信天道          |
| 重启清零   | 七日后世界重启                   | 所有势力疯抢归墟宁静区，世界走向末日临界          |

## 与主角机制的冲突（解释权之争）

- 世界意志要的是：人们用玄学解释异常，从而接受异常
- 李无妄带来的是：把异常当生活问题解决，从而否定异常的存在意义
- 冲突本质：谁来定义“这件事是什么”——是天命灾厄，还是屋里返潮

## 结局走向（收束逻辑）

- 世界意志在终局被迫接受“正常化”：异常滤镜被扫掉，世界恢复可居住
- 外界仍把归墟当禁地：因为他们记得恐惧，但解释权已经换了

## 写作注意事项

- 不要让它像会讲话的“人格反派”；它更像系统、规则、校验提示
- 它的恐怖感来自“无处不在”和“让人自证无罪”的压迫

## 正文生成素材（可直接粘贴改写）

### 出场模板（“它出现”的写法）

模板 1（天灾试探）：

- 雨不是落下来的，是被按下来的。
- 伞撑开也没用，水会绕过布面钻进骨缝里，像在找人的恐惧。
- 城里开始有人说：“天道在看。”

模板 2（认知污染）：

- 街边的牌匾忽然开口，字一个个往外掉，像牙齿。
- 人们一边念咒一边说着自己都听不懂的话，却越说越安心。
- 传闻像霉一样爬满每个角落：你只要听见，就已经被写进故事里。

### 冲突触发器（3 条）

1. 它用天气、灾象、地裂逼归墟露出“异常波动”，但师尊只会安排收衣、堵漏、打桩，导致它的校验失败并升级手段。
2. 它利用传闻成真，让外界对归墟的恐惧自行增长，形成“越解释越恐怖”的滚雪球。
3. 它在重启前制造“宁静区门票”狂热，逼所有势力把归墟变成世界中心，从而压垮主角的日常节奏。
