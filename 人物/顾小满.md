# 顾小满

## 角色定位（一句话）

归墟宗三弟子；种田线与“生活升级”的推动器，把师尊的日常需求变成宗门扩张的最小单位。

## 基本信息（写作可用）

| 项目 | 内容 |
|------|------|
| 身份 | 归墟宗三弟子 |
| 出身 | 农家子弟（曾被当作“灾星/不祥”驱赶） |
| 外貌 | 皮肤偏黑、手有茧、眼神干净；衣着朴素 |
| 性格 | 认真、踏实、慢热；一旦认可就极倔 |
| 核心动机 | 想过安稳日子：有地种、有饭吃、有地方不被赶 |

## 核心反差（喜剧抓手）

### 她的脑补方式：朴素 + 命运感
- 师尊教她怎么育苗，她会理解成“定天命”
- 师尊让她除虫、施肥，她会理解成“镇灾厄、养龙脉”

### 脑补模板（高频可复用）
| 李无妄的行为（生活） | 顾小满的解读（天命） |
|---|---|
| “地要松一松” | 开脉醒土，唤醒沉睡的灵脉 |
| “虫多了，想办法” | 斩断灾厄因果，驱逐秽潮 |
| “先种耐活的” | 先立稳根基，再谈大势 |
| “菜熟了就吃” | 真道不虚，落在饭碗里才算 |

## 能力与作用（不写境界）

- 农事技能强：看天色、辨土性、记节气，能把“生活逻辑”落到细活上
- 宗门经营辅助：统计收成、安排劳作、改良院子与菜地布局
- 情绪锚点：她的存在让归墟更“像家”，对冲末日恐怖

## 剧情弧线（按分卷）

| 分卷 | 状态 | 关键变化 |
|---|---|---|
| 第二卷 | 新入门的农家弟子 | 把“活下去”落实成“把地种好” |
| 第三卷 | 种田线成熟 | 通过一季季收成体现归墟宁静区的稳定性 |
| 第四卷 | 末日里的生产者 | 在灭世背景下仍能维持“有饭吃”，形成强烈反差 |

## 关系（写作可用）

| 角色 | 关系关键词 | 互动要点 |
|---|---|---|
| 李无妄 | 师尊/生活老师 | 她最在意的是师尊一句“行了”“够吃了” |
| 红绫 | 保护者/前辈 | 红绫会把她当“归墟根基”护着 |
| 钱多多 | 物资协作 | 钱多多买种子与工具，她负责把它们变成收成 |
| 宋缺 | 饭菜反馈回路 | “好吃/不好吃”会直接影响她种什么 |

## 说话风格与口癖

- 话少但句句落地：“能种”“能活”“够吃”
- 被触动时会突然很郑重：“师尊说过……”

## 正文生成素材（可直接粘贴改写）

### 出场模板（首次登场）

模板 1（被赶出来的人）：
- 她背着一只破竹筐，站在归墟外缘，脚下的泥像随时会把人吞下去。
- 她不敢哭，哭了也没人管，只盯着那片看起来最“正常”的菜地。
- 院里有人抬头看她一眼，问的不是来历，是：“会种地吗？”

模板 2（第一次被认可）：
- 她蹲下去抓一把土，捻了捻，低声说：“这土还能活。”
- 李无妄点头：“能活就行，先把日子过起来。”

### 对话模板（直接套用）

- “能种。”
- “我不怕累。”
- “师尊说……先把饭吃上。”
- “我会把地守住。”
- “够吃了就行，不贪。”

### 冲突触发器（3 条）

1. 外界把归墟作物当灵药抢夺，她把“护菜地”当护家，第一次正面顶住修士与诡异的双重压力。
2. 世界意志用“虫灾/霉潮/倒伏”试探归墟，她按师尊教的生活法子处理，结果外界把她当“镇灾命格”追捧。
3. 宋缺做饭失控导致食材需求失衡，她必须在“能吃”和“能活”之间调整种植，引出真实的生活矛盾。

## 禁写清单

- 不要把她写成天赋型战力；她的爽点是把小事做稳、把日子做实
- 不要让她过度脑补到抢红绫的钱多多的戏；她的脑补更温和、更贴近生活
