# 核心指令（Critical Instructions）

1.  **必须使用中文**：所有生成的内容（包括正文、对话、描述）必须是简体中文。
2.  **必须先读取设定文件**：
    - 默认先读取：`人物/红绫.md`、`人物/李无妄.md`（若存在）。
    - 优先读取 `章纲/` 下对应的章纲文件（如果存在），严格按章纲执行。
3.  **必须生成文件**：
    - 生成的小说章节必须保存为 `.txt` 文件。
    - **文件命名格式**：`草稿/卷名/XXXX.章节名.txt` (注意：必须写入草稿目录，等待人工审核)
      - 示例：`草稿/第一卷_归墟有个正常人/0002.这石头，压酸菜正好.txt`
      - 若卷目录不存在则创建。
      - 章节序号顺延。
    - 使用 `apply_patch` 或 `Write` 生成。
    - **文件内容约束**：
      - **禁止**在文件第一行写入章节标题（如“0005.收废品的胖子”）。
      - **禁止**在文件末尾写入字数估算（如“（本章字数估算：2750）”）。
      - 文件内容只包含正文。
4.  **内容格式规范**：
    - **纯文本格式**：不要使用 Markdown 的标题语法。
    - **段落清晰**：段落之间空一行。
5.  **输出要求**：
    - 在对话回复中输出：
      1.  章节标题
      2.  简要概述
      3.  **字数估算：XXXX**（必须在回复中单独一行显示）
      4.  文件路径
    - **禁止**在对话中直接输出全文。

## 🎯 番茄爆款创作原则

### 1. 爽点公式（核心）

**主角的降维打击（生活动作）** + **配角的过度脑补（恐怖解读）** = **极致的反差爽**

- **主角视角**：只是在喝水、修墙、喂狗、除草。心态：“这日子没法过了，怎么到处是害虫。”
- **外界视角**：这是混沌原浆、补天之石、上古神兽、斩断因果。心态：“大恐怖！不可名状！”

### 2. 单章节奏标尺（情绪推拉）

每章（约 2200-2600 字）必须包含完整的五段式节奏：

1.  **铺垫（生活烦恼）**：主角因为缺米/漏雨/吵闹而烦躁。（建立共情）
2.  **冲突（外界找茬）**：反派/危机逼近，气势汹汹，看似无解。（拉高期待）
3.  **解决（降维打击）**：主角为了解决生活烦恼，顺手把危机处理了。（释放爽点）
4.  **脑补（震惊全场）**：配角/反派自行脑补主角的深意，吓得跪地。（强化人设）
5.  **钩子（下章悬念）**：新的麻烦出现，或者刚才的动作引发了更大的后果。（留存率）

### 3. 黄金三章特例（如果是前三章）

- **第一章**：必须出现**灭世级追杀** vs **浇花/洗脚**。结尾必须杀一个大能。
- **第二章**：必须确立**弟子脑补**模式。白开水 = 神液。
- **第三章**：必须**震动外界**。一锄头下去，万里外宗门崩塌。

- **❌ 严禁主角觉醒**：李无妄永远不能知道自己很强。他必须认为：
  - 运气好：“那人怎么自己摔死了？”
  - 科学：“这是热胀冷缩原理。”
  - 对方弱：“这年头的修仙者身体素质真差。”
- **❌ 严禁恐怖降级**：外界描写必须**克苏鲁/恐怖/绝望**。反派越强、越诡异，主角一巴掌拍死才越爽。
- **❌ 严禁配角降智**：配角不是傻，是**信息差**。在他们的视角里，主角的行为完全符合“隐世禁忌”的特征。
