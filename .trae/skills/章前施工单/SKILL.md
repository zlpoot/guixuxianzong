# 章前施工单

## 目标

在不直接写正文的前提下，把已有大纲、设定与上一章结尾，整理成“本章可执行的写作施工单”，用于驱动后续的正文生成。
**核心逻辑：** 遵循“番茄爆款逻辑”，确保每一章都有明确的“生活目标 vs 外界阻碍”冲突。

## 何时调用

- 用户说“要写下一章/开新章/先给章纲/先出本章计划”
- 用户给了卷纲或上一章内容，要求“这一章怎么写更爽/更稳”
- 用户强调需要爆点、节奏、章末钩子、生活动作链约束

## 必读上下文（尽量从仓库读取）

1. `大纲.md`（核心依据，特别是“章前施工单”部分）
2. `章纲/` 下本章对应的章纲文件（如果存在）
3. `正文/` 下上一章的结尾（用于接钩子）
4. `设定/`、`人物/` 中与本章相关的条目

## 输出格式（Markdown 表格 + 补充说明）

请直接输出以下表格，不要废话：

### 1. 核心施工表

| 检查项                 | 内容                                              | 备注                                              |
| :--------------------- | :------------------------------------------------ | :------------------------------------------------ |
| **本章生活目标**       | (必填) 例如：我要修补院墙的那个洞。               | 必须极其庸俗、日常。                              |
| **外界阻碍（反派）**   | (必填) 例如：魔尊正在攻打世界壁垒（就是那个洞）。 | 必须宏大、恐怖。                                  |
| **主角动作（金手指）** | (必填) 例如：拿快干水泥糊上去，顺便拍平。         | 必须是物理层面的生活动作。                        |
| **脑补反馈（爽点）**   | (必填) 例如：魔尊被“混沌息壤”封印，永世不得超生。 | 弟子/反派的震惊反应。                             |
| **资源变化**           | (必填) 例如：水泥-1，威望+10000。                 | 必须有得失（扣除生活物资，获得声望/因果）。       |
| **章末钩子**           | (必填) 例如：刚才糊墙掉了一块泥，掉到下界会怎样？ | 吸引读者看下一章（四选一：来查/来求/天灾/榜单）。 |

### 2. 节奏与细节填充

- **单章节奏规划**：
  - **铺垫（20%）**：{生活烦恼与日常琐事}
  - **冲突（30%）**：{外界找茬与危机逼近}
  - **解决（30%）**：{生活动作降维打击}
  - **脑补（10%）**：{全场震惊与自我攻略}
  - **钩子（10%）**：{下章悬念}
- **关键金句/口头禅**：{列出主角本章要说的 1-2 句“凡尔赛”台词}
- **风险提示**：{是否有主角觉醒风险？配角是否降智？}

## 约束

- **严禁主角觉醒**：主角必须真诚地认为自己在做农活。
- **严禁配角降智**：配角的恐惧必须真实，逻辑必须自洽。
- **生活动作优先**：解决问题必须用“修、洗、扫、砸、晒、煮、钉”等动作。
