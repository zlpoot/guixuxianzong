# 人物卡维护

## 目标

把角色从“印象”固化成可执行的角色卡：动机、目标、底线、资源、口吻、成长线与关系网，支持连载长期一致。
**核心逻辑：** 遵循“番茄爆款人设”，强调**标签化**、**极致化**和**脑补滤镜**。

## 何时调用

- 用户新增角色或引入新势力核心人物
- 用户感觉角色“写飘了/前后不一致/台词不像他”
- 用户希望 AI 扩写时保持人物口吻与动机稳定

## 必读上下文

1. `人物/角色名.md`（若存在先读取并在其基础上增量更新）
2. 涉及该角色的最近章节内容（用于校准口吻与行为）
3. `大纲.md`（特别是“人设与脑补逻辑”部分）

## 输出与落地

- 先在对话中输出“变更摘要”（新增了什么、修正了什么）
- 然后用 `apply_patch` 创建或更新 `人物/角色名.md`

## 人物卡模板（写入文件）

请按以下 Markdown 格式写入文件：

```markdown
# {角色名}

## 1. 核心标签（番茄爆款风）
*   **身份**：{如：大弟子 / 反派魔尊}
*   **核心人设**：{一句话概括，如：暴力狂 / 极致财迷}
*   **脑补滤镜**：{核心误解逻辑，如：把一切看作剑意 / 把一切看作商机}
*   **口头禅**：{最具辨识度的一句话}

## 2. 误读机制（爽点来源）
*   **对主角的误解**：{他认为主角在做什么？}
*   **行为模式**：{脑补后他通常会做什么离谱的事？}
    *   *示例：* 主角倒水 -> 他认为是赐予神液 -> 当场跪下喝干 -> 突破境界

## 3. 基础档案
*   **外貌**：{3-5个关键词，如：红衣、冷艳、背剑}
*   **性格**：{优点与缺点的极致表现}
*   **恐惧点**：{他最怕什么？通常是怕主角生气/怕失去机缘}

## 4. 关系网
*   **对李无妄**：{崇拜/恐惧/依赖}
*   **对其他弟子**：{竞争/合作/鄙视}

## 5. 成长线（本卷）
*   **起点**：{刚出场时的状态}
*   **拐点**：{因为主角的某个无心之举而改变}
*   **终点**：{本卷结束时的状态}
```

## 约束
- **标签要极致**：不要写“性格复杂”，要写“极度贪财但怕死”。
- **脑补要逻辑自洽**：每个角色的脑补逻辑必须是固定的，不能今天觉得是剑意，明天觉得是厨艺（除非他是多重人格）。
