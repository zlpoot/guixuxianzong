# 力量体系设定

## 境界划分

### 传统修仙境界
从低到高依次是：炼气、筑基、金丹、元婴、化神。

| 境界 | 说明 |
|------|------|
| 炼气 | 初入修仙，凝聚灵气 |
| 筑基 | 铸造道基，寿元延长 |
| 金丹 | 凝结金丹，实力大增 |
| 元婴 | 破丹成婴，可御剑飞行 |
| 化神 | 修仙界巅峰，连归墟核心区都只能撑三息 |

### 特殊指标

#### 污染度
- 衡量被道诡侵蚀的程度
- 0%：纯净无污染
- 50%：开始出现轻微异化特征
- 80%：理智严重受损，行为异常
- 100%：彻底异化为道诡

#### 理智值
- 衡量心智稳定的程度
- 李无妄：100%（满值，永不下降）
- 普通修士：会因接触污染而下降
- 理智值归零 = 彻底异化

## 特殊能力

### 李无妄的能力

#### 绝对正常
- 任何诡异靠近他都会自动合理化
- 在他眼中，道诡只是"雾霾"、"流浪狗"、"皮肤病"
- 污染无法侵蚀他

#### 物理修正
- 用普通物理手段"修正"诡异
- 例1：扫帚扫地 → 扫掉高阶污染雾气
- 例2：菜刀削皮 → 剥离道诡法则
- 例3：石头砸"长虫" → 镇杀天灾级道诡
- 例4：给"流浪狗"洗澡 → 净化上古神兽

#### 认知过滤
- 自动将诡异事物转化为日常认知
- 天空血肉 → 雾霾
- 精神尖啸 → 耳鸣
- 污染触手 → 香肠/痤疮

### 弟子们的"能力"

#### 脑补（红绫、钱多多）
- 将主角的日常行为解读为深奥大道
- 例：白开水 → 无上圣水
- 例：削土豆 → 剥离大道法则的剑意
- 例：红烧肉 → 浓缩灵丹妙药

#### 灾难体质（双长老）
- 宋缺：火候失控，成品往往是法则洗礼
- 古百毒：炼丹必炸，炸即神通
