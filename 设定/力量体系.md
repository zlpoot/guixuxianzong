# 《归墟宗主只想种菜》力量体系设定

## 一、修仙界境界划分（外界视角）

本世界采用标准修仙体系，但在克苏鲁背景下，境界越高，受到的精神污染越重，越容易陷入疯狂。

| 境界       | 特征描述                                       | 寿元       | 战力表现                                           |
| :--------- | :--------------------------------------------- | :--------- | :------------------------------------------------- |
| **练气期** | 引气入体，身体初现异变（轻微畸变）。           | 100岁      | 凡人眼中的武林高手，可御使低阶法器。               |
| **筑基期** | 铸造道基，灵识初开，能感知到世界的“低语”。     | 200岁      | 御剑飞行，真火炼丹。                               |
| **金丹期** | 凝结金丹（往往带有诡异花纹），初涉法则。       | 500岁      | 即使肉身损毁，金丹亦可夺舍重生。                   |
| **元婴期** | 破丹成婴，元婴往往呈现非人形态（触手/复眼）。  | 1000岁     | 瞬息千里，移山填海，是一方老祖级别。               |
| **化神期** | 元神寄托虚空，开始接触世界本质（不可名状）。   | 3000岁     | 掌控天地之力，言出法随（由于疯狂，往往语无伦次）。 |
| **炼虚期** | 虚实转换，身体可在物质与能量间切换。           | 5000岁     | 撕裂空间，遨游虚空。                               |
| **合体期** | 肉身与元神合一，返璞归真（或彻底异化）。       | 万年       | 滴血重生，不死不灭（除非被更高层次抹除）。         |
| **大乘期** | 渡劫前的最后阶段，半只脚踏入仙界（或归墟）。   | 与天地同寿 | 此界巅峰，一念生世界，一念灭星辰。                 |
| **渡劫期** | 引动天劫，飞升仙界（或坠入归墟成为古神眷属）。 | -          | 渡过则成仙，失败则灰飞烟灭或沦为怪物。             |

## 二、主角眼中的世界（李无妄视角）

主角没有任何修为（自认为），他在归墟种菜，外界的一切对他来说都是“大自然的干扰”。

| 外界境界   | 主角眼中的对应物（参照系）                              | 主角应对方式                              |
| :--------- | :------------------------------------------------------ | :---------------------------------------- |
| **练气期** | **蚊子/苍蝇**                                           | 随手拍死，或者喷点杀虫剂（普通水）。      |
| **筑基期** | **大号蟑螂/老鼠**                                       | 一脚踩死，或者拿扫帚赶走。                |
| **金丹期** | **野猫/野狗**                                           | 吓唬一下，扔块石头（翻天印）砸跑。        |
| **元婴期** | **比较凶的野兽（狼/野猪）**                             | 拿锄头（开天神斧）敲晕，或者设个陷阱。    |
| **化神期** | **大型猛兽（老虎/狮子）**                               | 需要认真对待，比如关门放狗（大黄）。      |
| **炼虚期** | **自然灾害（暴雨/狂风）**                               | 骂骂咧咧，赶紧收衣服/关窗户（封印空间）。 |
| **合体期** | **严重的自然灾害（地震/洪水）**                         | 叹气，修补房屋（重塑世界规则）。          |
| **大乘期** | **不可抗力（陨石/核爆，但在主角看来是家里煤气罐炸了）** | 极其愤怒，因为炸坏了锅（混沌至宝）。      |
| **渡劫期** | **天道找茬（物业来催水电费）**                          | 跟它讲道理，或者直接把物业打一顿。        |

| :------------ | :------------------------------------- | :------------------------------------------- |

| **混沌之气**  | 触之即死的毁灭能量，万物归一的本源。   | **雾霾/灰尘**。太脏了，得经常打扫。          |
| **大道法则**  | 维持世界运转的根本规则，不可触碰。     | **蜘蛛网**。这破房子怎么到处都是网，烦死了。 |

| **无上丹药**  | 起死回生，白骨生肉，凡人服之立地成仙。 | **糖豆/零食/剩饭**。大黄都不爱吃。           |
| **灵兽/神兽** | 血脉高贵，统御万妖，毁天灭地。         | **家禽/宠物/食材**。养肥了过年吃肉。         |

## 四、力量体系使用规范（给AI的指令）

1.  **严禁混淆视角**：
    - 描写主角时，必须坚持“生活视角”，绝对不能出现“灵力波动”、“威压”、“神识”等修仙词汇。
    - 描写配角/外界时，必须坚持“修仙视角（带克苏鲁风）”，强调恐惧、不可名状、脑补。
2.  **战力崩坏预警**：
    - 哪怕是“大乘期”修士，在主角面前也只能是“稍微大一点的麻烦”。
    - 主角解决问题的方式必须是**非战斗的**（如：洗衣服、做饭、打扫卫生）。
    - 绝对不能让主角“爆发小宇宙”或“觉醒前世记忆”。
3.  **关于“大乘期”**：
    - 书中可以出现“大乘期”强者，但必须作为**极度稀有**的背景板或大Boss出现。
    - 前期（前100章）建议最高只出现到**化神期**或**炼虚期**，保留战力升级空间。
    - 如果第二章出现了“大乘期”描述，请确认为**夸张修辞**（如“堪比大乘期威压”），或者该物品等级确实极高（如混沌神液），足以让大乘期疯狂。
