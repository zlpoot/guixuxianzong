# 0008.种点啥好呢

## 1. 章前施工单（核心）

| 检查项 | 内容 |
| :--- | :--- |
| **本章生活目标** | 规划新的菜地，想种点反季蔬菜。 |
| **外界阻碍（反派）** | 天道察觉到归墟规则异常，降下“天劫雷罚”试图修正。 |
| **主角动作（金手指）** | 觉得天太阴（雷云），不利于作物生长，拿出一面镜子（昊天镜）反射阳光。 |
| **脑补反馈（爽点）** | 昊天镜光柱直冲云霄，直接把劫云冲散。天道意志再次被“物理劝退”。 |
| **资源变化** | 阳光+1，天道威严-1。 |
| **章末钩子** | 钱多多提议把种出来的菜拿到山下去卖，试探市场。 |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（铺垫）：** 李无妄拿着种子（蟠桃核）犹豫，觉得光照不足。
2.  **镜头二（冲突）：** 天空乌云密布，紫雷滚滚，仿佛末日降临。
3.  **镜头三（解决）：** 李无妄：“这鬼天气，给点阳光行不行？” 拿出镜子调整角度。光柱刺破乌云，瞬间晴空万里。
4.  **镜头四（脑补）：** 红绫和钱多多：师尊在与天争锋！逆天改命！
5.  **镜头五（收束）：** 种下种子。李无妄：“希望能活吧。”

## 3. 信息与伏笔
*   **新增信息**：主角的日常用品全是神器。
*   **伏笔埋藏**：这颗种子长出来后会引发更大的动静。
