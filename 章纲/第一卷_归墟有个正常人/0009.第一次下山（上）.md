# 0009.第一次下山（上）

## 1. 章前施工单（核心）

| 检查项 | 内容 |
| :--- | :--- |
| **本章生活目标** | 带着种好的大白菜，下山去集市换点生活用品。 |
| **外界阻碍（反派）** | 山下集市被黑恶势力（猛虎帮）霸占，强收保护费。 |
| **主角动作（金手指）** | 推着独轮车（混沌战车），无视路障和陷阱，直接碾过去。 |
| **脑补反馈（爽点）** | 猛虎帮帮主（金丹期）看到独轮车碾碎了防御阵法，吓得不敢动弹。 |
| **资源变化** | 独轮车耐久度-0，猛虎帮路障-1。 |
| **章末钩子** | 来到集市，发现没人敢买他的菜（因为菜上散发着恐怖的道韵）。 |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（铺垫）：** 李无妄整装待发，红绫和钱多多乔装打扮（扮成随从）。
2.  **镜头二（冲突）：** 猛虎帮设卡拦截，叫嚣收过路费。
3.  **镜头三（解决）：** 李无妄：“让让，车刹不住了！” 独轮车轰隆隆碾过，路障粉碎。
4.  **镜头四（脑补）：** 帮主：那车轮上刻着毁灭道纹！不可阻挡！
5.  **镜头五（收束）：** 顺利进城。李无妄：“这城里怎么冷冷清清的？”

## 3. 信息与伏笔
*   **新增信息**：主角下山，地图切换到凡人城镇（其实是修仙坊市）。
*   **伏笔埋藏**：集市里隐藏着第三个弟子。
