# 002 这不是白开水，是混沌神液

## 1. 章前施工单（核心）

| 检查项                 | 内容                                                                                          |
| :--------------------- | :-------------------------------------------------------------------------------------------- |
| **本卷表面主线**       | 攒钱过冬（修屋顶+囤粮），顺便解决终身大事。                                                   |
| **本章生活目标**       | 把救回来的人（红绫）弄醒，别死在院子里（晦气）。                                              |
| **外界阻碍（反派）**   | 红绫重伤濒死，经脉寸断，普通丹药无效。                                                        |
| **主角选择**           | 不把人丢出去，但先把话说死：救命可以，醒了就打工抵债，别搞什么“拜师收徒”。                    |
| **主角动作（金手指）** | 打了桶井水烧开，加盐做成“盐水”，强行给红绫灌下去。                                            |
| **脑补反馈（爽点）**   | 红绫喝下后，经脉像被重新“洗过”一样复原，认定这是“混沌神液/归墟灵泉”，师尊是在用大道给她续命。 |
| **资源变化**           | 盐-1勺，获得大弟子一枚（虽然脑子不太好使）。                                                  |
| **代价/后果**          | 红绫从此把“干活抵债”理解成“入门考验”，越认真越偏；她的突破气息也开始向外泄露。                |
| **章末钩子**           | 红绫发誓效忠，李无妄让她“去扫地”（其实是真扫地），她理解为“扫除心魔”。                        |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（落差）：**
    - _细节：_ 李无妄把红绫拖进屋（像拖麻袋），一路滴血把地板糟蹋得一塌糊涂。他一边擦地一边嘀咕：“这得用多少水……”
2.  **镜头二（交易）：**
    - _细节：_ 他蹲在红绫旁边，像看一件麻烦事：“你要死别死我家。醒了就干活抵债，别跟我扯什么拜师。”红绫昏迷中却听成“入门誓言”，心里一热。
3.  **镜头三（解决）：**
    - _细节：_ 李无妄去打井水、烧开、加盐，端着缺口破碗捏着鼻子灌下去：“喝点盐水，省得请大夫。”
4.  **镜头四（错位）：**
    - _细节：_ 红绫意识回笼，感觉体内经脉像被清洗、缝合、重塑。她看着那桶井水、那只破碗，眼前却是“混沌初开、灵泉涌动”的异象。
5.  **镜头五（拉扯）：**
    - _细节：_ 红绫跪求拜师，话还没说完，李无妄先抛下一句：“要跪去外头跪，地刚擦完。”红绫更是笃定：前辈嫌她心不净，不许污了洞天福地。
6.  **镜头六（钩子）：**
    - _细节：_ 李无妄把扫帚往她怀里一塞：“活在那儿，扫干净。”红绫抱着扫帚如抱仙剑，立誓“扫尽心魔”。她突破后的气息在归墟边缘一闪即逝，引来远处天象轻动。

## 3. 信息与伏笔

- **新增信息**：红绫的脑补机制（把生活用品看作神器）。
- **伏笔埋藏**：红绫的突破会引来天劫，为下一章埋雷。
- **伏笔回收**：无。
