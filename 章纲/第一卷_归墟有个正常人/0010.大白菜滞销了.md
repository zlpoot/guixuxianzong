# 0010.大白菜滞销了

## 1. 章前施工单（核心）

| 检查项 | 内容 |
| :--- | :--- |
| **本章生活目标** | 推销大白菜。 |
| **外界阻碍（反派）** | 识货的炼丹师想买但不敢靠近（道韵太强），不识货的凡人嫌贵。 |
| **主角动作（金手指）** | 现场切开一颗白菜试吃。 |
| **脑补反馈（爽点）** | 切开瞬间，香气（丹气）化龙，全城修士突破瓶颈。 |
| **资源变化** | 白菜-1，全城修士境界+1。 |
| **章末钩子** | 一个落魄的小乞丐（顾小满）挤过人群，想讨一片菜叶子吃。 |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（铺垫）：** 李无妄吆喝：“新鲜大白菜，两文钱一斤！” 众人围观，无人敢买。
2.  **镜头二（冲突）：** 某炼丹大师路过，惊呼“这是神药”，但被道韵震慑，无法靠近。
3.  **镜头三（解决）：** 李无妄：“怎么？怕不好吃？我切一个给你们尝尝。” 菜刀落下，异象陡生。
4.  **镜头四（脑补）：** 满城飘香，闻一口延年益寿。众人疯狂。
5.  **镜头五（收束）：** 小乞丐出现。李无妄看着她，心生怜悯。

## 3. 信息与伏笔
*   **新增信息**：引入三弟子顾小满（气运担当）。
*   **伏笔埋藏**：顾小满吃下菜叶后，身上的封印开始松动。
