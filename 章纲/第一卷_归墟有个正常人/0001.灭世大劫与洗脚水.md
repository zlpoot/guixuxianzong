# 001 灭世大劫与洗脚水

## 1. 章前施工单（核心）

| 检查项                 | 内容                                                               |
| :--------------------- | :----------------------------------------------------------------- |
| **本卷表面主线**       | 攒钱过冬（修屋顶+囤粮），顺便解决终身大事。                        |
| **本章生活目标**       | 给后院的白菜浇水，顺便洗个脚。                                     |
| **外界阻碍（反派）**   | 血煞魔尊（化神期）追杀前朝公主红绫，试图用“血海大阵”炼化方圆百里。 |
| **主角选择**           | 不开门不出院子，先把菜和脚处理完，外面的动静当野猫。               |
| **主角动作（金手指）** | 嫌外面太吵（魔尊狂笑），以为是野猫叫，把洗脚水泼出墙外。           |
| **脑补反馈（爽点）**   | 魔尊眼中的“混沌神水”从天而降，血海瞬间被净化，魔尊当场湮灭。       |
| **资源变化**           | 白菜水分+1，洗脚水-1，意外救下大弟子。                             |
| **代价/后果**          | 血煞宗命牌尽碎，后续必有人来探归墟；红绫把这里当救命宗门赖上。     |
| **章末钩子**           | 红绫苏醒，看到李无妄拿着“神器”（瓢），产生极致误会。               |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（冲突）：**
    - _细节：_ 开篇即高潮。红衣女子（红绫）浑身是血，跌撞到归墟山门前。身后血海翻涌，血煞魔尊狂笑：“今日谁也救不了你！”血海大阵开始收拢，连山门的青石都被腐蚀得滋滋作响。
2.  **镜头二（铺垫）：**
    - _细节：_ 画面切到墙内。李无妄挽着裤腿给白菜浇水，嘴里念叨“省着点吃、攒着点用”。他眉头紧锁，不是因为外面天塌了，而是因为“这白菜有点蔫”——再蔫下去就得花钱买菜，过冬更难。
3.  **镜头三（解决）：**
    - _细节：_ 墙外魔尊祭出最强杀招“血魔解体”，血海化作一张巨口扑向山门。墙内李无妄洗完脚，端起洗脚水，嫌外头又吵又臭，嘟囔着“这野猫叫得真烦人”，顺手把洗脚水从墙头泼了出去，打算顺便冲冲墙根的泥。
4.  **镜头四（脑补）：**
    - _细节：_ 在红绫眼中，那不是洗脚水，那是蕴含大道法则的“净世天河”！水珠落下，血海像被天光照到的雪一样迅速消融。魔尊前一刻还在狂笑，下一刻脸色惨白，连惨叫都来不及发出，连同血海大阵一起湮灭。
5.  **镜头五（收束）：**
    - _细节：_ 一切归于平静。李无妄甩甩手：“总算清净了。”转身回屋睡觉。红绫却像从鬼门关被拎回来，盯着那扇普通的木门，跪地磕头：“多谢前辈救命之恩！”
6.  **镜头六（钩子）：**
    - _细节：_ 红绫昏迷前最后一个念头是：前辈明明在院内，却能隔墙净世，归墟果然是禁忌之地。远处血煞宗命牌接连碎裂，宗门大殿里一片死寂。

## 3. 信息与伏笔

- **新增信息**：归墟是禁忌之地；李无妄是这里的“普通人”。
- **伏笔埋藏**：魔尊虽死，但其背后的宗门（血煞宗）会发现命牌破碎，引出后续试探。
- **规则提示**：外界凶煞越靠近归墟，越像被“洗掉”一样无声消解；墙内日常反而更像“天威”的触发点。
- **伏笔回收**：无。
