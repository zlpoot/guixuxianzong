# 0005.收废品的胖子

## 1. 章前施工单（核心）

| 检查项                 | 内容                                                                                     |
| :--------------------- | :--------------------------------------------------------------------------------------- |
| **本卷表面主线**       | 攒钱过冬（修屋顶+囤粮），顺便解决终身大事。                                              |
| **本章生活目标**       | 把院子里的“废品”卖掉换米。                                                               |
| **外界阻碍（反派）**   | 钱多多的追兵（元婴期）不死心，试图闯入归墟搜索。                                         |
| **主角选择**           | 为了尽快换到米，宁愿“亏本”送出一件看着最破的旧物。                                       |
| **主角动作（金手指）** | 为了展示“废品”质量，随手拿起一个“破罐子”（吞天魔罐）演示：“你看这玩意儿虽然旧，但能装。” |
| **脑补反馈（爽点）**   | 钱多多看到主角把漫天毒瘴（追兵放的毒）全装进了罐子，吓得哆嗦。                           |
| **资源变化**           | 破罐子-1，获得银票若干（在主角眼里是钱，在外界是灵石兑换券）。                           |
| **代价/后果**          | 罐子离开归墟，等于把“宝光坐标”带去外界，后续会引来魔道与妖族的连锁追逐。                 |
| **章末钩子**           | 钱多多决定留下当“代理人”，要把这些神物运作到外界去。                                     |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（铺垫）：** 李无妄热情推销：“小胖子，你看这口锅（炼妖壶），炖肉特别香。”
2.  **镜头二（冲突）：** 追兵在院外放毒，试图逼出钱多多。毒气弥漫，草木枯黄，连院里的白菜都开始打蔫。
3.  **镜头三（解决）：** 李无妄：“怎么起雾了？还挺呛人。” 拿起“破罐子”对着空中一晃，漫天毒气瞬间被吸干。
4.  **镜头四（脑补）：** 钱多多：吞天噬地！这是传说中的魔道至宝！师尊竟然拿来当吸尘器？
5.  **镜头五（收束）：** 李无妄：“这罐子送你了，换你身上那袋米，行不行？” 钱多多疯狂点头。

## 3. 信息与伏笔

- **新增信息**：确立钱多多的“变现”功能；外界手段开始影响院内日常（毒气伤菜）。
- **伏笔埋藏**：那个罐子被钱多多带出去后，会引发魔道的争夺。
