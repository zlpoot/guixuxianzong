# 0131.包办婚姻要不得
## 1. 章前施工单（核心）
| 检查项 | 内容 |
| :--- | :--- |
| **本章生活目标** | 担心那个所谓的“仙界未婚妻”找上门来。 |
| **外界阻碍（反派）** | 仙界的确有人想通过接引使者的通道下来，但那是为了追杀使者，并非什么未婚妻。 |
| **主角动作（金手指）** | 觉得家里大门不安全，决定把“后门”（飞升通道）给堵上。搬了一块“破石头”（镇界碑）压在井口（通道口）。 |
| **脑补反馈（爽点）** | 通道被镇界碑彻底封死。仙界追兵撞在碑上，头破血流，以为是绝世大能的拒绝。 |
| **资源变化** | 破石头-1，安全感+1。 |
| **章末钩子** | 通道堵住了，但还有点漏风（灵气外泄）。 |

## 2. 细纲（剧情流）
- **(1) 只有我受伤的世界**
  - 主角在院子里踱步，越想越觉得“未婚妻”这事儿不靠谱。
  - “现在的诈骗手段太高明了，连仙人都骗。”
  - 他决定未雨绸缪，不能让陌生人随便进来。
- **(2) 堵门行动**
  - 主角来到后院的枯井旁（接引使者就是从这里爬出来的）。
  - 他觉得这井口阴森森的，就把那块平时用来压咸菜缸的“大青石”（镇界碑）搬了过来。
  - “咣当”一声，盖在了井口上。
- **(3) 仙界的惨案**
  - 视角切到飞升通道另一头。
  - 一群金甲仙兵正顺着通道极速下潜，准备捉拿“叛徒”接引使者。
  - 突然，通道尽头出现一块巍峨的石碑，散发着镇压诸天的恐怖气息。
  - 冲在最前面的仙将直接撞了上去，肉身崩碎，元神重创。
- **(4) 迪化时刻**
  - 仙兵惊恐后撤：“这是……太古镇界碑？！下界怎会有此等神物？”
  - “难道那位存在不喜我等打扰？”
  - 众人不敢造次，只能隔着通道跪拜请罪。
- **(5) 还是有点漏**
  - 主角拍拍手，觉得石头盖得不严实，还有风从缝隙里吹出来（其实是仙气倒灌）。
  - “得找点泥糊上。”

  - 主角去池塘边挖了一桶黑泥。这泥重得离谱，一桶有千斤重（混沌息壤）。
  - 他把洗脚水倒进去搅拌，泥巴变得粘稠又坚固。

  - 红绫：“师尊用的那是……息壤？！传说中女娲补天的神物？”

  - 众弟子跪地膜拜，不敢出声打扰这神圣的仪式。
- **(4) 彻底封死**
  - 主角把泥巴糊在石头缝里，抹平，压实。
  - “这下连苍蝇都飞不进来了。”
  - 随着最后一道缝隙被堵住，仙凡通道彻底断绝。
- **(5) 抬头看天**
  - 干完活，主角擦擦汗，抬头看天。
  - 发现天空中有一朵云形状很奇怪，像个窟窿（其实是天门映照）。
  - “屋顶要是漏了更麻烦。”

  - 他拿出一个“大喇叭”（由某种神兽喉骨制成），对着天空喊话。
  - “楼上的！有没有公德心啊！关窗！”
- **(3) 言出法随**
  - 声音化作金色波纹，直冲九霄。
  - 天道法则仿佛听到了不可违抗的命令，原本洞开的天门开始强行闭合。
- **(4) 悲催的天子**
  - 此时，一位身穿华服的青年（天子）正一只脚跨出天门。
  - 突然天门关闭，巨大的规则之力挤压而来。
  - “啊——！谁在坑我？！”
  - 天子被卡在两界之间，不上不下，尴尬至极。
- **(5) 终于安静了**
  - 主角听到“砰”的一声（天门关上的声音），满意地点点头。
  - “现在的邻居，素质真差，非得吼两句才听话。”
