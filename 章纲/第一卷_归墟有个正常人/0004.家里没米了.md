# 0004.家里没米了

## 1. 章前施工单（核心）

| 检查项                 | 内容                                                                         |
| :--------------------- | :--------------------------------------------------------------------------- |
| **本卷表面主线**       | 攒钱过冬（修屋顶+囤粮），顺便解决终身大事。                                  |
| **本章生活目标**       | 发现米缸空了，准备解决吃饭问题。                                             |
| **外界阻碍（反派）**   | 万宝阁少阁主钱多多被家族内斗追杀，携带重宝逃入归墟边缘。                     |
| **主角选择**           | 不出门买米，先把院子里堆积的“破烂”清出去换口粮。                             |
| **主角动作（金手指）** | 看着空米缸叹气，决定把院子里堆积的“破烂”（神器）清理一下，看能不能换点米。   |
| **脑补反馈（爽点）**   | 钱多多误入归墟，被主角随手扔出的“垃圾”（神器碎片）砸中，惊为天人，纳头便拜。 |
| **资源变化**           | 破烂-1，钱多多（二弟子）+1。                                                 |
| **代价/后果**          | “废品变现”这条线一旦启动，归墟的神物会被持续运出，势必引来各方觊觎。         |
| **章末钩子**           | 钱多多看着满院子的“垃圾”双眼放光，引发主角误会（以为他是收废品的）。         |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（铺垫）：** 红绫做饭，发现米缸见底。李无妄愁眉苦脸：“这日子没法过了，得想办法搞点钱。”
2.  **镜头二（冲突）：** 归墟外围，钱多多浑身是血，怀揣万宝阁至宝“聚宝盆”，身后是家族执法队的追杀。
3.  **镜头三（解决）：** 李无妄整理杂物，随手把一块“磨刀石”（断裂的诛仙剑尖）甩到墙外当垫脚石。剑尖遇到杀气自发“归位”，顺势斩穿追杀者的飞舟，飞舟当场解体。
4.  **镜头四（脑补）：** 钱多多看着那块“磨刀石”，感受到其中蕴含的恐怖剑意，认定这是隐世大能在救他。
5.  **镜头五（收束）：** 钱多多爬进院子，李无妄看着这个胖子，问：“收废品吗？这堆破烂便宜卖你。”

## 3. 信息与伏笔

- **新增信息**：引入二弟子钱多多（商业担当），万宝阁背景。
- **伏笔埋藏**：钱多多带来的“聚宝盆”其实是主角当年丢弃的饭碗。
