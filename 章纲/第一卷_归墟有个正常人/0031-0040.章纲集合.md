# 0031.宗门大比的请柬

## 1. 章前施工单（核心）

| 检查项                 | 内容                                                                                           |
| :--------------------- | :--------------------------------------------------------------------------------------------- |
| **本章生活目标**       | 收到一张“花里胡哨”的纸（请柬），以为是广告单。                                                 |
| **外界阻碍（反派）**   | 正道联盟使者前来送“宗门大比”请柬，态度傲慢，想给归墟一个下马威。                               |
| **主角动作（金手指）** | 看着请柬上的金粉嫌脏，随手用来垫了垫茶壶底。“这纸挺厚实，吸水不错。”                           |
| **脑补反馈（爽点）**   | 使者看到代表正道荣耀的请柬被用来垫茶壶，且茶壶散发的道韵直接压碎了请柬上的禁制，吓得不敢说话。 |
| **资源变化**           | 请柬（茶垫）+1，正道威严-1。                                                                   |
| **章末钩子**           | 钱多多看到请柬，惊呼这是十年一度的盛会，有很多商机。                                           |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（铺垫）：** 使者御剑而来，声势浩大，要在归墟显摆。
2.  **镜头二（冲突）：** 递上请柬，言语中暗讽归墟是不入流的野鸡宗门。
3.  **镜头三（解决）：** 李无妄：“哦，放那吧。” 顺手把刚烧开的茶壶放上去。滋——请柬冒烟。
4.  **镜头四（脑补）：** 使者：那茶壶是九天玄铁？那请柬可是有元婴期禁制的啊！瞬间烧穿？
5.  **镜头五（收束）：** 李无妄：“这纸质量不行，烫坏了桌子怎么办？”

## 3. 信息与伏笔

- **新增信息**：开启“宗门大比”副本。
- **伏笔埋藏**：李无妄决定带弟子去“见见世面”（其实是被钱多多忽悠去摆摊）。
