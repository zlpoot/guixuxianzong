# 0007.大扫除

## 1. 章前施工单（核心）

| 检查项 | 内容 |
| :--- | :--- |
| **本章生活目标** | 院子里杂草（灵药）太多，需要除草。 |
| **外界阻碍（反派）** | 药王谷的人感应到归墟有惊世药气，前来探查，试图强抢。 |
| **主角动作（金手指）** | 嫌杂草长得快，拿镰刀（死神之镰）一顿乱割。 |
| **脑补反馈（爽点）** | 药王谷长老看到主角把“九转还魂草”当杂草割了喂猪，心痛到吐血，道心崩溃。 |
| **资源变化** | 杂草-1堆，药王谷长老的三观-1。 |
| **章末钩子** | 药王谷长老跪求“杂草”，李无妄大方赠送，长老痛哭流涕。 |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（铺垫）：** 李无妄抱怨：“这雨后草长得太快了，把我的菜都挡住了。”
2.  **镜头二（冲突）：** 药王谷众人气势汹汹，准备破阵采药。
3.  **镜头三（解决）：** 李无妄挥舞镰刀，割草如麻。每一刀下去，都斩断了灵草的生机（在药王谷眼里），但在主角眼里只是除草。
4.  **镜头四（脑补）：** 药王谷长老：他在斩断因果！他在演示枯荣之道！
5.  **镜头五（收束）：** 长老跪求枯草。李无妄：“拿去拿去，省得我扔了。” 长老如获至宝。

## 3. 信息与伏笔
*   **新增信息**：归墟里的“杂草”都是外界的圣药。
*   **伏笔埋藏**：这些枯草被药王谷带回，炼出了惊世神丹。
