# 003 一锄头，震碎了千万里

## 1. 章前施工单（核心）

| 检查项                 | 内容                                                                                                     |
| :--------------------- | :------------------------------------------------------------------------------------------------------- |
| **本卷表面主线**       | 攒钱过冬（修屋顶+囤粮），顺便解决终身大事。                                                              |
| **本章生活目标**       | 修补被红绫突破时震裂的院墙。                                                                             |
| **外界阻碍（反派）**   | 血煞宗老祖（魔尊师父）通过秘法窥探归墟，准备跨界打击。                                                   |
| **主角选择**           | 不追究红绫闯祸，先把墙补好，免得冬天漏风。                                                               |
| **主角动作（金手指）** | 拿起锄头夯土（修墙），用力砸了几下地基。                                                                 |
| **脑补反馈（爽点）**   | 血煞宗隔空咒杀刚触到归墟边界就崩溃反噬，主角夯土的动静却被外界误读为“天道落槌”，血煞宗护山大阵当场震裂。 |
| **资源变化**           | 墙修好了，威望+MAX（外界认定归墟是禁地）。                                                               |
| **代价/后果**          | 外界把归墟的“禁忌等级”抬高，引来更多试探与趋利者，钱多多因此盯上这里。                                   |
| **章末钩子**           | 血煞宗宣布封山百年，外界传闻“归墟有大恐怖”。有人来查（万宝阁钱多多登场）。                               |

## 2. 剧情细纲（镜头语言）

1.  **镜头一（起因）：**
    - _细节：_ 红绫前一夜突破把院墙震裂出一条细缝，风一吹就呜呜响。李无妄摸着裂缝直叹气：“这要是冬天漏风，得多烧多少柴。”
2.  **镜头二（对照）：**
    - _细节：_ 画面切到血煞宗。老祖盯着命牌碎末，怒到发笑，摆出“血河大阵”，以秘法牵引因果，要把“杀我徒儿之人”隔空咒杀。
3.  **镜头三（触界）：**
    - _细节：_ 咒力化作一道血线，循着因果一路钻向归墟。血线刚碰到归墟边缘，像被无形的清水一冲，瞬间散开，反噬回卷，阵中长老齐齐喷血。
4.  **镜头四（解决）：**
    - _细节：_ 墙内李无妄完全不知情，举锄夯土：“给我实诚点，别再裂了。”锄头落下那一下，归墟边界随之“沉了一沉”，外界的人却像听到天鼓，护山大阵咔嚓一声裂开一条大口子。
5.  **镜头五（脑补）：**
    - _细节：_ 红绫抱着扫帚站在旁边，听到那一声“沉”，只觉得心神都要碎：“师尊不是在修墙，是在把天道钉回正位！”
6.  **镜头六（收束）：**
    - _细节：_ 李无妄拍拍手：“行了，过冬不漏风了。”血煞宗那边却以为自己惹到了禁忌，仓促封山百年。消息传到万宝阁，钱多多眼睛一亮：“禁地里还能过日子？那是真生意。”

## 3. 信息与伏笔

- **新增信息**：李无妄的物理动作可以无视距离打击因果；归墟“墙内日常”会被外界误读成天威。
- **伏笔埋藏**：钱多多的到来将开启“经营线”。
- **伏笔回收**：回收了第1章魔尊之死的伏笔（打了小的来了老的，老的也被打服了）。
