# 《归墟仙宗》全书创作大纲

## 一、项目概述

- **核心主题：** 正常即异常。主角是混乱世界中唯一的“理性锚点”。
- **叙事风格：** 轻松幽默 + 克苏鲁式诡异 + 生活化叙事。
- **主角状态：** 全程不觉醒（或极晚觉醒）。真诚地认为自己是普通人。
- **冲突来源：** 世界试图同化主角 vs 他人对主角的过度解读。
- **力量体系：** 模糊化。强调“污染度”（越高越诡异）与“理智值”（主角唯一满值）。

## 三、剧情大纲（分卷规划）

### **第一卷：归墟有个正常人（1-50 万字）**

- **核心目标：** 建立世界观，确立“认知错位”基调，组建核心团队。
- **关键事件：**
  1.  **归墟苏醒：** 李无妄醒来，身处高污染区。他认为只是“大雾天气”，决定打扫卫生。
  2.  **大弟子归位：** 红绫濒临异化躲入归墟。李无妄给她一杯白开水（圣水），她污染清零。红绫脑补：宗主为我洗髓！
  3.  **二弟子入伙：** 钱多多来推销“护身符”。李无妄用后院石头磨了个镇纸抵债。钱多多发现石头能镇压邪祟，倒贴钱成为弟子。
  4.  **神兽看门：** 怪狗大黄闯入。李无妄给它洗澡喂食。大黄恢复神兽形态，但只在李无妄面前是狗。
  5.  **双长老争端（高潮）：** 宋缺与古百毒因“地盘之争”在宗门外引发灾难（火与毒）。李无妄路过皱眉：“烟熏到我的菜了。”
      - **结果：** 两人瞬间停手，争相加入归墟宗赎罪。
      - **误解：** 弟子们认为宗主一句话就镇压了两位灾难巨头。
  6.  **卷末悬念：** 世界意志首次注意到这个“异常点”，降下一道“天灾”试探。李无妄以为是“暴雨”，让大家收衣服。

### **第二卷：正常也是一种污染（51-100 万字）**

- **核心目标：** 扩大地图，外界势力介入，双长老功能化，主角的“正常”被神话。
- **关键事件：**
  1.  **外出采购：** 归墟宗缺瓦片。李无妄带队下山。弟子们全副武装，结果诡异生物闻到李无妄气息纷纷避让。
  2.  **双长老切磋：** 李无妄建议“灶台丹炉合并省柴火”。宋缺与古百毒被迫合作，一个炸坑一个火烧，瞬间填平。
      - **误解：** 外界视为“阴阳大阵”，归墟宗威慑力大增。
  3.  **说书风波：** 李无妄在茶馆听说书（讲的是自己的传说），以为是虚构小说，点评“逻辑不通”。听客们震惊：宗主在暗示我们泄露天机！
  4.  **污染入侵：** 世界意志尝试同化归墟宗。物品变异（茶壶长嘴）。李无妄认为“家具老化”，强行物理修正（锯平、敲回）。
      - **效果：** 物理修正法则，变异物品恢复正常。
  5.  **卷末悬念：** 世界意志发现物理手段无效，决定从“认知”层面入手，制造一个“完全理性”的假主角来挑战李无妄。

### **第三卷：理智的孤岛（101-150 万字）**

- **核心目标：** 深化冲突，主角面临真正的认知危机，弟子与长老成长。
- **关键事件：**
  1.  **假主角登场：** 世界意志化身“天道之子”，拥有绝对逻辑，试图证明李无妄是错的。两人进行“道争”（其实是生活辩论）。
  2.  **生活化道争：** 对方说“世界是虚妄的”，李无妄说“菜熟了得吃”。对方无法反驳，因为李无妄吃的菜真的能饱腹。**真实感击败虚无感。**
  3.  **长老危机：** 古百毒力量失控即将自爆，宋缺没有逃跑，用“万火”包裹试图中和。
      - **转折：** 李无妄递给他们一杯水：“喝口水，降降火。”两人力量平息。他们意识到彼此不再是累赘，而是宗门的“双翼”。
  4.  **世界排斥：** 世界开始排斥归墟宗，地块开始下沉。李无妄认为“地基不稳”，打桩加固。归墟宗被他强行“钉”在了世界壁垒上。
  5.  **卷末悬念：** 世界意志决定掀桌子，启动“灭世重启”，将所有异常清零，包括李无妄。

### **第四卷：唯一的答案（151-200 万字）**

- **核心目标：** 最终决战，升华主题，结局收束。
- **关键事件：**
  1.  **灭世降临：** 天空崩塌，万物异化。人类陷入疯狂。
  2.  **最后日常：** 李无妄不管外面末日，坚持要喝完最后一杯茶，种完最后一块地。他的“日常”成为了风暴眼中的“宁静区”。
  3.  **信徒集结：** 受过恩惠的人、被吓服的诡异、弟子们，全部守在归墟宗外，用生命阻挡世界意志的侵蚀。
  4.  **终极修正：** 世界意志本体降临。李无妄把它当成“最大的害虫”，拿起扫帚进行了一次“大扫除”。
      - **效果：** 扫帚扫掉了世界的“异常滤镜”。
  5.  **结局（开放式·日常永恒）：** 世界意志被“正常”同化，世界变得正常了。但外人眼中的归墟宗依旧诡异。
  6.  **尾声：** 李无妄坐在门口喝茶。
      - 路人 A：“那就是救世主住的禁地吗？”
      - 路人 B：“嘘，别吵，听说他在参悟大道。”
      - 李无妄：“哎，这茶叶又涨价了。”

## 四、核心看点与场景设计

### 1. “双长老”的竞争美学

- **场景：** 宗门聚餐。
- **过程：** 宋缺端上焦黑的饼，古百毒端上爆炸过的粥。两人互相瞪眼，都希望宗主吃自己做的。
- **主角反应：** “都行，别浪费。”
- **后果：** 弟子们为了争夺“宗主吃剩的残渣”大打出手。双长老认为自己的作品得到了认可，关系缓和了一秒钟，下一秒又开始争“谁的残渣更有营养”。

### 2. “物理修正”法则

- **场景：** 诡异生物入侵。
- **过程：** 一只长满触手的怪物试图吞噬李无妄。
- **主角反应：** “这章鱼成精了？正好缺个拖把。”
- **后果：** 李无妄把触手剪下来绑在棍子上。怪物因无法理解“被做成工具”而逻辑崩溃，消散。李无妄得到一把好用的拖把。

### 3. “模糊化”的长老形象

- **描写技巧：** 不写皱纹、白发。写“身影在烟火中忽隐忽现”，“声音仿佛来自四面八方”。
- **目的：** 保持神秘感，让读者聚焦于他们的能力失控，而非身份背景。

## 五、编辑写作建议（避坑指南）

1.  **主角的“钝感力”：**
    - 李无妄的心理活动必须**纯粹**。不能有“我这样做他们肯定会吓到”的想法。
    - 例如：看到弟子吐血，他想的是“这孩子上火”，而不是“我功力太深”。
2.  **长老关系的“界限”：**
    - 宋缺与古百毒**严禁**出现暧昧描写。
    - 他们的互动必须是“专业互怼”。例如：“你的火太弱了”vs“你的药太臭了”。
    - 羁绊在于“只有你能承受我的灾难”，这是一种**战友般的共生**，而非爱情。
3.  **恐怖与幽默的平衡：**
    - 描写诡异时，多用侧面描写（弟子的恐惧、环境的扭曲）。
    - 主角在场时，氛围必须瞬间转为**温馨田园风**。这种反差是本书的核心爽点。
4.  **节奏把控：**
    - 每 3 章一个小误会，每 10 章一个中等事件，每 50 章一个大高潮。
    - 不要让误会无限叠加，偶尔要让主角解决一些实际困难（如真的没米了），增加生活质感。
5.  **关于觉醒：**
    - 建议**全程不觉醒**。一旦主角意识到自己很强，那种“无辜的强大感”就会消失，变成普通的无敌文。
    - 如果必须觉醒，放在大结局最后一章，且觉醒后他依然选择继续种菜。
